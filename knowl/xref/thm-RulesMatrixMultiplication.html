Theorem 2.1.6. Matrix Multiplication Rules.

Suppose \(A\text{,}\) \(B\text{,}\) and \(C\) are matrices that are the right size for the products to be defined. And suppose \(I\) is the identity matrix that is the right size as well. The following properties hold:
  1. Identity Property: \(IA = A\) and \(AI = A\)
    🔗
    🔗
  2. Associativity: \(A(BC) = (AB)C\)
    🔗
    🔗
  3. (Left) Distributive Property: \(A(B + C) = AB + AC\)
    🔗
    🔗
  4. (Right) Distributive Property: \((A + B)C = AC + BC\)
    🔗
    🔗
  5. Regrouping with Scalar Multiplication; For any scalar \(c\text{,}\) \(c(AB) = (cA)B = A(cB)\)
    🔗
    🔗
  6. Transpose of a Product: \((AB)^T = B^T A^T\)
    🔗
    🔗
