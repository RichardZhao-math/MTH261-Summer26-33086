    Section 2.3 Characterizations of Invertible Matrices

    Up to this point, we have explored various concepts seemingly in isolation: pivot positions, linear independence, span, one-to-one and onto transformations, and matrix equations. In this section, we will bring all of these ideas together under one unified framework: the Invertible Matrix Theorem. Instead of treating these as a list of separate rules to memorize, we will build connections between them from the ground up. You will see how discovering just one piece of information about a square matrix instantly unlocks a wealth of knowledge about its fundamental structure and the linear transformation it defines.
    🔗

    Objectives

    After this section, students will be able to:
    🔗
    • relate the core properties of square matrices (such as pivot positions, linear independence, span, and trivial solutions) directly to the concept of invertibility.
      🔗
      🔗
    • construct logical chains of equivalence to justify and prove various components of the Invertible Matrix Theorem.
      🔗
      🔗
    • apply the Invertible Matrix Theorem to quickly deduce the characteristics of a matrix or a system of equations without performing exhaustive row operations.
      🔗
      🔗
    • define an invertible linear transformation and formally link it to the inverse of its standard matrix.
      🔗
      🔗
    🔗
    It is super important to be able to tell whether a matrix is invertible. One of the most important theorems in linear algebra is the Invertible Matrix Theorem, which identifies several conditions that are equivalent to a matrix being invertible.
    🔗

    Theorem 2.3.1. Invertible Matrix Theorem.

    Let \(A\) be an \(n \times n\) matrix. The following statements are equivalent.
    1. \(A\) is invertible.
      🔗
      🔗
    2. \(A\) is row equivalent to \(I_n\text{,}\) the \(n \times n\) identity matrix.
      🔗
      🔗
    3. \(A\) has \(n\) pivot positions.
      🔗
      🔗
    4. \(A\v{x} = \v{0}\) has only the trivial solution.
      🔗
      🔗
    5. The columns of \(A\) are linearly independent.
      🔗
      🔗
    6. The linear transformation \(x \mapsto A\v{x}\) is one-to-one.
      🔗
      🔗
    7. \(A\v{x} = \v{b}\) is consistent for all \(\v{b}\in \R^n\text{.}\)
      🔗
      🔗
    8. The columns of \(A\) span \(\R^n\text{.}\)
      🔗
      🔗
    9. The linear transformation \(\v{x} \mapsto A\v{x}\) maps \(\R^n\) onto \(\R^n\text{.}\)
      🔗
      🔗
    10. There is an \(n \times n\) matrix \(C\) such that \(CA = I\text{.}\)
      🔗
      🔗
    11. There is an \(n \times n\) matrix \(D\) such that \(AD = I\text{.}\)
      🔗
      🔗
    12. \(A^T\) is invertible.
      🔗
      🔗
    🔗
    🔗
    This big theorem that ties everything we have learned together! If you Google the Invertible Matrix Theorem, there are a lot more equivalent conditions than what we have listed here. Some of them are just rephrasing the conditions using other terminology and some of them requires a bit more linear algebra concepts that we are building towards. We will certainly circle back to this theorem and add more conditions when we learn more!
    🔗
    Your textbook includes a blurb to prove this theorem (by directing us back to various places to find the proofs). Instead of proving them (they are super straightforward!), let’s discuss the method to prove such a big theorem!
    🔗
    Proving two statements are equivalent is the same thing as proving a biconditional statement. Let’s say we have two statements, \(P\) and \(Q\text{,}\) then "they are equivalent" means both "if \(P\) then \(Q\)" and "if \(Q\) then \(P\)". Symbolically, we use the notation "\(\implies\)" for implication and "\(\iff\)" for equivalence. So we are proving both \(P \implies Q\) and \(Q \implies P\)
    🔗
    To prove the invertible matrix theorem, we want to establish a chain of equivalence. That is, we want to link all the statements together using the equivalence notation \(\iff\text{.}\)
    🔗
    Let’s start small! Let’s say we have three statements \(P\text{,}\) \(Q\text{,}\) and \(R\text{.}\) To prove that they are all equivalent, we need to prove that \(P \iff Q \iff R\text{.}\) One way to do so is to prove (1) \(P \implies Q\text{,}\) (2) \(Q \implies R\text{,}\) and (3) \(R \implies P\text{.}\) This creates a continuous logical loop! If we assume any one of these statements is true, we can follow the chain of implications to guarantee that the others are also true. Because we can travel from any single statement to every other statement in this cycle, they must all share the exact same truth value.
    🔗
    You will prove the invertible matrix theorem by establishing the chain of implication in Lab Report #3.
    🔗

    Example 2.3.2.

    Let \(A = \begin{bmatrix} 2 \amp 0 \amp 0 \amp 0 \\ 1 \amp 3 \amp 0 \amp 0 \\ 8 \amp -2 \amp 1 \amp 0 \\ 9 \amp 12 \amp -3 \amp -3 \end{bmatrix}\text{.}\) We can actually learn a lot by the invertible matrix theorem.
    🔗
    One can very quickly determine that \(A\) is invertible (because the \(\det A \neq 0\)). Alternatively, we can solve the homogeneous equation quickly that \(A\v{x} = \v{0}\) has only the trivial solution.
    🔗
    And we can draw so many conclusions by the fact that \(A\) is invertible using the invertible matrix theorem! A couple examples are:
    1. \(A\) can be row reduced to a \(4 \times 4\) identity matrix;
      🔗
      🔗
    2. \(A\) has \(4\) pivot positions;
      🔗
      🔗
    3. The columns of \(A\) are linearly independent;
      🔗
      🔗
    4. The columns of \(A\) spans \(\R^4\text{;}\)
      🔗
      🔗
    5. The equation \(A\v{x} = \v{b}\) is always consistent for every \(\v{b} \in \R^4\text{;}\)
      🔗
      🔗
    6. The linear transformation \(\v{x} \mapsto A\v{x}\) is injective,
      🔗
      🔗
    And the list goes on!
    🔗
    Recall these are the things that would have taken us hours to figure out before when we didn’t know about the invertible matrix theorem!
    🔗
    🔗
    Recall all the linear transformations we learn from this class can be represented using matrix transformations. That is, we can define the linear transformation \(T: \R^m \to \R^n\) by \(T(\v{x}) = A\v{x}\text{.}\)
    🔗
    If \(A\) is invertible, then what linear transformation does \(A^{-1}\v{x}\) define?
    🔗
    We see that multiplication of \(A^{-1}\) undoes the linear transformation \(T\text{,}\) so the inverse matrix defines the inverse of a linear transformation!
    🔗

    Definition 2.3.4.

    A linear transformation \(T: \R^n \to \R^n\) is invertible if there exists a function \(S: \R^n \to \R^n\) such that
    \begin{align*} S\lp T(x) \rp \amp \v{x} \amp\amp \forall \v{x} \in \R^n \end{align*}
    and
    \begin{align*} T\lp S(x) \rp \amp \v{x} \amp\amp \forall \v{x} \in \R^n \end{align*}
    🔗
    🔗

    Theorem 2.3.5.

    Let \(T: \R^n \to \R^n\) be a linear transformation with the standard matrix \(A\text{.}\) Then \(T\) is invertible if and only if \(A\) is an invertible matrix.
    🔗
    In this case, the linear transformation \(S\) given by \(S(\v{x}) = A^{-1}\v{x}\) is the unique function satisfying the invertible definition for \(T\) and is called the inverse of \(T\text{.}\)
    🔗
    🔗

    Example 2.3.6.

    Let’s say \(T: \R^n \to \R^n\) is an injective linear transformation. What can we say about \(T\text{?}\)
    🔗
    \(T\) being injective means that the columns of \(A\text{,}\) the standard matrix that defines this linear transformation, are linearly independent. This implies that \(A\) has \(n\) pivot positions.
    🔗
    By the invertible matrix theorem, we know that \(A\) is invertible, and hence, \(T\) is an invertible linear transformation!
    🔗
    Also by the invertible matrix theorem, the columns of \(A\) spans the codomain, \(\R^n\text{,}\) and hence \(T\) must be surjective.
    🔗
    See how powerful the invertible matrix theorem is! Once we know just a bit of information about the matrix/linear transformation, the invertible matrix theorem will do the rest!
    🔗
    🔗

    Checkpoint 2.3.7.

    Let \(T: \R^n \to \R^n\) be a surjective linear transformation. What can we say about \(T\text{?}\)
    🔗
    🔗
    Fancy terminology! A transformation that are both injective and surjective is called a bijection. An immediate result we can get is that a transformation is invertible if and only if it is a bijection!
