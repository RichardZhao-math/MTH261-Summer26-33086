    Section 4.2 The Characteristic Equation

    Now that we know how to find the eigenvectors corresponding to a specific eigenvalue, the next question is: How do we find the eigenvalues of a matrix?
    🔗
    We can find the eigenvalues of a matrix by, again, observing this equation
    \begin{equation*} (A - \lambda I)\v{x} = \v{0} \end{equation*}
    Recall that eigenvectors are non-trivial solutions to this homogeneous equation, so this equation must have infinitely many solutions. That is, the matrix \(A - \lambda I\) must be singular (not invertible).
    🔗
    We can say a lot about a non-invertible matrix. The fact that will be super useful to us is that a matrix is non-invertible if and only if its determinant is zero. So we can find all eigenvalues of a matrix by solving the equation \(\det(A - \lambda I) = 0\text{.}\)
    🔗

    Example 4.2.1.

    Let \(A = \begin{bmatrix} 3 \amp 2 \\ -2 \amp -2 \end{bmatrix}\) and we want to find all the eigenvalues of this matrix.
    🔗
    Let \(\lambda\) be an eigenvalue of \(A\text{.}\) Then we want the matrix \(A - \lambda I\) be non-invertible to guarantee non-trivial solutions to the equation \((A - \lambda I)\v{x} = \v{0}\text{.}\) That is, we want the \(\lambda\) values such that \(\det(A - \lambda I) = 0\text{.}\)
    🔗
    Let’s start by computing the determinant of the matrix \(A - \lambda I\text{:}\)
    \begin{align*} \det(A - \lambda I) \amp= \begin{vmatrix} 3 - \lambda \amp 2 \\ -2 \amp -2 - \lambda \end{vmatrix} \\ \amp= (3 - \lambda)(-2 - \lambda) - (2)(-2) \\ \amp= -6 - 3\lambda + 2\lambda + \lambda^2 + 4 \\ \amp= \lambda^2 - \lambda - 2 \end{align*}
    This is essentially a quadratic equation in terms of \(\lambda\) and we can totally solve it! Factoring the quadratic, we have
    \begin{equation*} (\lambda - 2)(\lambda + 1) = 0 \qquad \implies \quad \lambda = 2 \quad \text{or} \quad \lambda = -1 \end{equation*}
    Hence, the eigenvalues of the matrix \(A\) are \(2\) and \(-1\text{.}\)
    🔗
    🔗

    Checkpoint 4.2.2.

    Find the eigenvalues of the matrix \(B = \begin{bmatrix} 1 \amp 1 \\ 1 \amp 1 \end{bmatrix}\text{.}\)
    🔗
    🔗
    Since the equation \(\det(A - \lambda I) = 0\) captures the characteristics of the matrix \(A - \lambda I\) that this matrix must be non-invertible, we call this equation the characteristic equation of the matrix \(A\text{.}\)
    🔗

    Definition 4.2.3. Characteristic Equations.

    Let \(A\) be an \(n \times n\) matrix. The equation
    \begin{equation*} \det (A - \lambda I) = 0 \end{equation*}
    is called the characteristic equation of the matrix \(A\text{.}\) A scalar \(\lambda\) is an eigenvalue of \(A\) if and only if it is a solution to the characteristic equation of \(A\text{.}\)
    🔗
    The characteristic equation will always be a polynomial equation, so \(\det(A - \lambda I)\) is called the characteristic polynomial of the matrix \(A\text{.}\)
    🔗
    🔗
    In practice, finding the eigenvalues of an \(n \times n\) matrix results in solving a polynomial equation of degree \(n\text{.}\) This is almost always difficult as \(n\) gets larger. So we will leave this to computers, except in the \(2 \times 2\) case and some easy \(3 \times 3\) cases.
    🔗

    Example 4.2.4.

    Let \(A = \begin{bmatrix} 1 \amp 3 \amp 3 \\ -3 \amp -5 \amp -3 \\ 3 \amp 3 \amp 1 \end{bmatrix}\text{.}\) What are the eigenvalues of this matrix?
    🔗
    We can find the eigenvalues by solving the characteristic equation \(\det(A - \lambda I) = 0\text{,}\) so let’s start by computing the determinant of the matrix \(A - \lambda I\text{:}\)
    \begin{align*} \det(A - \lambda I) \amp= \begin{vmatrix} 1 - \lambda \amp 3 \amp 3 \\ -3 \amp -5 - \lambda \amp -3 \\ 3 \amp 3 \amp 1 - \lambda \end{vmatrix} \\ \amp= (1 - \lambda) \begin{vmatrix} -5 - \lambda \amp -3 \\ 3 \amp 1 - \lambda \end{vmatrix} - 3 \begin{vmatrix} -3 \amp -3 \\ 3 \amp 1 - \lambda \end{vmatrix} + 3 \begin{vmatrix} -3 \amp -5 - \lambda \\ 3 \amp 3 \end{vmatrix} \\ \amp= (1 - \lambda)((-5 - \lambda)(1 - \lambda) + 9) - 3((-3)(1 - \lambda) + 9) + 3(-9 - 3(-5 - \lambda)) \\ \amp= (1 - \lambda)(\lambda^2 + 4\lambda + 4) - 3(3\lambda + 6) + 3(-9 + 15 + 3\lambda) \\ \amp= (1 - \lambda)(\lambda + 2)^2 - 3(3\lambda + 6) + 3(3\lambda + 6) \\ \amp= (1 - \lambda)(\lambda + 2)^2 \end{align*}
    Since \(\det (A - \lambda I)\) is already factored, we can easily solve the characteristic equation \(\det(A - \lambda I) = 0\) and find the eigenvalues:
    \begin{equation*} (1 - \lambda)(\lambda + 2)^2 = 0 \qquad \implies \qquad \lambda = 1 \quad \text{or} \quad \lambda = -2 \end{equation*}
    Hence, the eigenvalues of the matrix \(A\) are \(1\) and \(-2\text{.}\)
