    Section 2.2 The Inverse of a Matrix

    In our journey through linear algebra, we have explored how to multiply matrices and how to represent systems of equations as the matrix equation \(A\v{x} = \v{b}\text{.}\) A natural mathematical question arises: can we "undo" matrix multiplication? In the realm of real numbers, we solve algebraic equations like \(ax = b\) by multiplying by the reciprocal (or inverse) of \(a\text{.}\) In this section, we investigate whether a similar structural concept exists for matrices.
    🔗

    Objectives

    After this section, students will be able to:
    🔗
    • find the inverse of a \(2 \times 2\) invertible matrix using a formula;
      🔗
      🔗
    • find the inverse of an invertible matrix using the Gaussian Elimination algorithm;
      🔗
      🔗
    • solve systems of linear equations using matrix inverses;
      🔗
      🔗
    • construct elementary matrices and its inverse for each elementary row operation;
      🔗
      🔗
    • express any invertible matrix, and its inverse, as a product of elementary matrices.
      🔗
      🔗
    🔗

    Investigation 2.2.1.

    Let \(A = \begin{bmatrix} 0 \amp 2 \\ 1 \amp -2 \end{bmatrix}\text{.}\) Find a matrix \(B\) such that \(AB = I\) where \(I\) is the \(2 \times 2\) identity matrix.
    🔗
    Hint.
    There are two possible strategies:
    🔗
    Strategy #1: Guess and check! This strategy is doable for a \(2 \times 2\) matrix but usually quite tedious. The matrix \(A\) is made special so that it isn’t too bad at all.
    🔗
    Strategy #2: Remember how matrix multiplication works: When we multiply \(A\) by \(B\text{,}\) each column of the answer is found by multiplying the corresponding column of \(B\) by \(A\text{.}\)
    🔗
    So we know that if \(\v{b}_1\) is the first column of \(B\text{,}\) then we will need \(A\v{b}_1\) to be \(\begin{bmatrix} 1 \\ 0 \end{bmatrix}\) since that is the first column of the identity matrix. If we call the entries of that column \(x\) and \(y\text{,}\) then we can solve a system of equations to figure out what \(x\) and \(y\) are!
    🔗
    Then we can do a similar process to figure out what the second column of \(B\) is!
    🔗
    🔗
    Solution 1. Using Strategy #1...
    You probably discovered that the only way to get the \(1\) in the top left of the answer is to put a \(\frac{1}{2}\) in the bottom left of \(B\text{.}\) Then you can figure out that you need a \(1\) in the top left of \(B\) to get the \(0\) in the bottom left of the answer.
    🔗
    Now you need a \(0\) in the top right of the answer. because of the \(0\) in \(A\text{,}\) your answer for this spot will be \(2\) times whatever you put in the bottom right of \(B\text{.}\) So that has to be a \(0\text{.}\)
    🔗
    Finally that \(0\) in the bottom right of \(B\) means that you need a \(1\) in your top right spot in order to get the \(1\) you need in the bottom right of the identity matrix. So the matrix \(B = \begin{bmatrix} 1 \amp 1 \\ \frac{1}{2} \amp 0 \end{bmatrix}\)
    🔗
    🔗
    Solution 2. Using Strategy #2...
    If we call our first column of \(B\) \(\v{b}_1 = \begin{bmatrix} x \\ y \end{bmatrix}\text{,}\) then we know that we need
    \begin{equation*} \begin{cases} 0x + 2y = 1 \\ 1x - 2y = 0 \end{cases} \end{equation*}
    We can solve this using the substitution method easily since the top equation tells us \(y = \frac{1}{2}\) and substituting this into the second equation gives us
    \begin{equation*} x - 1 = 0 \qquad \implies \qquad x = 1 \end{equation*}
    So the first column of \(B\) will have to be \(\v{b}_1 = \begin{bmatrix} 1 \\ \frac{1}{2} \end{bmatrix}\text{.}\)
    🔗
    Now if we call our second column of \(B\) \(\v{b}_2 = \begin{bmatrix} x \\ y \end{bmatrix}\text{,}\) then we know that we need
    \begin{equation*} \begin{cases} 0x + 2y = 0 \\ 1x - 2y = 1 \end{cases} \end{equation*}
    This one is even quicker to solve because the top equation tells us \(y = 0\) and substituting this into the second equation gives us \(x = 1\text{.}\) So the second column of \(B\) will have to be \(\v{b}_2 = \begin{bmatrix} 1 \\ 0 \end{bmatrix}\text{.}\)
    🔗
    Therefore, the matrix \(B = \begin{bmatrix} 1 \amp 1 \\ \frac{1}{2} \amp 0 \end{bmatrix}\text{.}\)
    🔗
    And we can quickly verify that \(AB = I\text{:}\)
    \begin{equation*} \begin{bmatrix} 0 \amp 2 \\ 1 \amp -2 \end{bmatrix} \begin{bmatrix} 1 \amp 1 \\ \frac{1}{2} \amp 0 \end{bmatrix} = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} \end{equation*}
    🔗
    🔗
    Now that you found \(B\text{,}\) take a minute and compute \(BA\text{.}\)
    🔗
    Solution 3.
    \(BA\) is also \(I\text{!}\)
    🔗
    🔗
    🔗
    This matrix \(B\) is a special buddy of \(A\text{.}\) This is called the inverse of \(A\text{!}\)
    🔗

    Subsection Inverse of a Matrix

    Definition 2.2.1. Inverse of a Matrix.

    If \(A\) is a square matrix, a matrix \(B\) is called an inverse of \(A\) if and only if \(AB=BA=I\text{.}\) A matrix that has an inverse is called an invertible matrix.
    🔗
    🔗

    Example 2.2.2.

    An exmaple: \(A=\begin{bmatrix}0\amp 2\\ 1\amp -2\end{bmatrix}\) is invertible and \(A^{-1}=\begin{bmatrix}1\amp 1\\ \frac{1}{2}\amp 0\end{bmatrix}\) since...
    \begin{equation*} \begin{bmatrix}0\amp 2\\ 1\amp -2\end{bmatrix}\begin{bmatrix}1\amp 1\\ \frac{1}{2}\amp 0\end{bmatrix}=\begin{bmatrix}1\amp 1\\ \frac{1}{2}\amp 0\end{bmatrix}\begin{bmatrix}0\amp 2\\ 1\amp -2\end{bmatrix}=I \end{equation*}
    🔗
    A non-example: \(A=\begin{bmatrix}0\amp 0\\ 1\amp -2\end{bmatrix}\) is NOT invertible because no matter what matrix you try you will get a row of zeros:
    \begin{equation*} \begin{bmatrix}0\amp 0\\ 1\amp -2\end{bmatrix}\begin{bmatrix}a\amp b\\ c\amp d\end{bmatrix}=\begin{bmatrix}0\amp 0\\ \text{Doesn't}\amp \text{Matter}\end{bmatrix} \end{equation*}
    🔗
    🔗

    Theorem 2.2.3.

    A matrix can only have one inverse.
    🔗
    🔗

    Proof.
    Suppose \(B\) and \(C\) are both inverses of \(A\text{.}\) Then
    \begin{equation*} AB=I \qquad \text{ and } \qquad AC=I \end{equation*}
    This means \(AB=AC\text{.}\)
    🔗
    Multiplying both sides of equation (on the left) by \(B\text{,}\) we get
    \begin{equation*} B(AB)=B(AC) \end{equation*}
    Using the associative property, we get
    \begin{equation*} (BA)B=(BA)C \end{equation*}
    Since we also know that \(BA=I\text{,}\) then \(IB=IC\text{.}\)
    🔗
    So by the identity property, \(B=C\text{.}\)
    🔗
    🔗
    In the \(2 \times 2\) case, there is a quick way to find out if a matrix is invertible and there is a nice formula for the inverse. We just have to define two terms:
    🔗

    Definition 2.2.4. Determinant of \(2\times 2\) Matrices.

    Let \(A\) be a \(2 \times 2\) matrix. The determinant of the matrix \(A\) is defined by
    \begin{equation*} \det A = \det \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} = ad - bc \end{equation*}
    🔗
    🔗

    Definition 2.2.5. Adjugate of \(2\times 2\) Matrices.

    Let \(A\) be a \(2 \times 2\) matrix. The adjugate of the matrix \(A\) is
    \begin{equation*} \adj A = \begin{bmatrix} d \amp -b \\ -c \amp a \end{bmatrix} \end{equation*}
    The adjugate of \(A\) can be obtained by (1) swapping the elements in the main diagonal, and (2) changing the sign of the elements not on the main diagonal.
    🔗
    🔗
    The matrix \(A = \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}\) is invertible if and only if \(\det A \neq 0\text{.}\) And\(A^{-1}\) is
    \begin{equation*} A^{-1} = \frac{1}{\det A} \adj A = \frac{1}{\det A} \begin{bmatrix} d \amp -b \\ -c \amp a \end{bmatrix} \end{equation*}
    🔗

    Example 2.2.6.

    Let \(A = \begin{bmatrix} 0 \amp 2 \\ 1 \amp -2 \end{bmatrix}\text{.}\) Notice that
    \begin{equation*} \det A = (0)(-2) - (2)(1) = -2 \end{equation*}
    Since \(\det A \neq 0\text{,}\) we know that \(A^{-1}\) exists.
    🔗
    We can also find \(A^{-1}\text{.}\) We know that \(\adj A = \begin{bmatrix} -2 \amp -2 \\ -1 \amp 0 \end{bmatrix}\text{.}\) Then
    \begin{equation*} A^{-1} = \frac{1}{-2} \begin{bmatrix} -2 \amp -2 \\ -1 \amp 0 \end{bmatrix} = \begin{bmatrix} 1 \amp 1 \\ \frac{1}{2} \amp 0 \end{bmatrix} \end{equation*}
    🔗
    If you want to verify that \(A^{-1}\) is indeed the inverse of \(A\text{,}\) try multiplying them together and you should get back \(I\text{.}\)
    🔗
    🔗
    Remember how we worked so hard to solve systems of linear equations in week 1? And remember that every system of linear equations can be written as a Matrix Equation \(A \v{x} = \v{b}\) in Section 1.4? Well, if the coefficient matrix \(A\) is invertible, then we can solve this equation using \(A^{-1}\) just like back in Algebra 1!
    🔗
    Let’s assume that \(A\) is invertible, then \(A^{-1}A = I\text{.}\) Say we have a Matrix Equation \(A\v{x} = \v{b}\text{.}\) We certainly can multiply the same thing (say \(A^{-1}\)) on two sides of the equation. We get
    \begin{align*} A^{-1} \lp A\v{x} \rp \amp= A^{-1}\v{b} \\ \implies \qquad \lp A^{-1} A \rp \v{x} \amp= A^{-1}\v{b} \\ \implies \qquad \qquad \quad \, \, \v{x} \amp= A^{-1} \v{b} \end{align*}
    So the solution to this equation must be \(x\v{x} = A^{-1} \v{b}\text{.}\)
    🔗
    If you want to verify that this is indeed a solution, then we can plug in \(\v{x} = A^{-1}\v{b}\) into the left side of our equation. We see that
    \begin{equation*} A\v{x} = A\lp A^{-1} \v{b} \rp = \lp AA^{-1} \rp \v{b} = I\v{b} = \v{b} \end{equation*}
    So this really is a solution to the equation! Let’s make it a theorem!
    🔗

    Theorem 2.2.7.

    If \(A\) is an invertible matrix, then the Matrix Equation \(A\v{x} = \v{b}\) has exactly one solution. The solution is \(\v{x} = A^{-1} \v{b}\text{.}\)
    🔗
    🔗

    Example 2.2.8.

    Consider the following system of linear equations:
    \begin{equation*} \begin{cases} 3x + 2y = 5 \\ 5x - 8y = 14 \end{cases} \end{equation*}
    The Matrix Equation version of this system is \(A \v{x} = \v{b}\text{,}\) where
    \begin{equation*} A = \begin{bmatrix} 3 \amp 2 \\ 5 \amp -8 \end{bmatrix} \qquad \text{ and } \qquad \v{b} = \begin{bmatrix} 5 \\ 14 \end{bmatrix} \end{equation*}
    🔗
    Notice that
    \begin{equation*} \det A = 3(-8) - 2(5) = -34 \neq 0 \end{equation*}
    So \(A\) is invertible! We can find the solution using the little theorem we just came up with!
    🔗
    Since \(\adj A = \begin{bmatrix} -8 \amp -2 \\ -5 \amp 3 \end{bmatrix}\text{,}\) then we get
    \begin{equation*} A^{-1} = \frac{1}{-34} \begin{bmatrix} -8 \amp -2 \\ -5 \amp 3 \end{bmatrix} = \begin{bmatrix} \frac{4}{17} \amp \frac{1}{17} \\ \frac{5}{34} \amp \frac{-3}{34} \end{bmatrix} \end{equation*}
    By our theorem, the solution to the system is
    \begin{equation*} \v{x} = A^{-1}\v{b} = \begin{bmatrix} \frac{4}{17} \amp \frac{1}{17} \\ \frac{5}{34} \amp \frac{-3}{34} \end{bmatrix} \begin{bmatrix} 5 \\ 14 \end{bmatrix} = \begin{bmatrix} 2 \\ -\frac{1}{2} \end{bmatrix} \end{equation*}
    🔗
    🔗

    Checkpoint 2.2.9.

    Solve the following system of linear equation
    \begin{equation*} \begin{cases} 2x - 3y = 3 \\ 4x + 5y = 1 \end{cases} \end{equation*}
    by
    1. writing the system as a Matrix Equation;
      🔗
      🔗
    2. finding the inverse of the coefficient matrix;
      🔗
      🔗
    3. using the inverse to solve the system.
      🔗
      🔗
    🔗
    Solution.
    1. The matrix equation is \(\begin{bmatrix}2\amp -3\\ 4\amp 5\end{bmatrix}\begin{bmatrix}x\\ y\end{bmatrix}=\begin{bmatrix}3\\ 1\end{bmatrix}\)
    🔗
    2. The determinant is \(22\) and the adjugate matrix is \(\begin{bmatrix}5\amp 3\\ -4\amp 2\end{bmatrix}\text{,}\) so the inverse is
    \begin{equation*} \frac{1}{22}\begin{bmatrix}5\amp 3\\ -4\amp 2\end{bmatrix}=\begin{bmatrix}\frac{5}{22}\amp \frac{3}{22}\\ \frac{-4}{22}\amp \frac{2}{22}\end{bmatrix} \end{equation*}
    🔗
    3. The solution is \(\begin{bmatrix}\frac{5}{22}\amp \frac{3}{22}\\ \frac{-4}{22}\amp \frac{2}{22}\end{bmatrix}\begin{bmatrix}3\\ 1\end{bmatrix}=\begin{bmatrix}\frac{9}{11}\\ \frac{-5}{11}\end{bmatrix}\)
    🔗
    🔗
    🔗
    There is another way to compute inverse matrices efficiently, which is using the Gaussian Elimination algorithm we learned back in Section 1.2.
    🔗
    Suppose we want to find the inverse of a \(2 \times 2\) matrix \(\begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}\text{.}\) That is, we are looking for a \(2 \times 2\) matrix \(\begin{bmatrix} x_1 \amp x_2 \\ y_1 \amp y_2 \end{bmatrix}\) such that
    \begin{equation*} \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}\begin{bmatrix} x_1 \amp x_2 \\ y_1 \amp y_2 \end{bmatrix} = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} \end{equation*}
    Since we multiply these matrices by multiplying the \(\begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}\) by each of the column vectors \(\begin{bmatrix} x_1 \\ y_1 \end{bmatrix}\) and \(\begin{bmatrix} x_2 \\ y_2 \end{bmatrix}\text{,}\) the first matrix-vector product needs to give us the first column of \(I\) and the second matrix-vector product needs to give us the second column of \(I\text{.}\) That means that finding the inverse boils down to solving two systems of equations given by the matrix equations:
    \begin{equation*} \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} \begin{bmatrix} x_1 \\ y_1 \end{bmatrix} = \begin{bmatrix} 1 \\ 0 \end{bmatrix} \qquad \text{ and } \qquad \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} \begin{bmatrix} x_2 \\ y_2 \end{bmatrix} = \begin{bmatrix} 0 \\ 1 \end{bmatrix} \end{equation*}
    Observe that these two systems of linear equations have the same coefficient matrix. So if we used augmented matrices, we would end up doing the same steps to put the matrices into reduced row echelon form.
    🔗
    The technique based on this idea is to put these two equations into ONE giant augmented matrix:
    \begin{equation*} \begin{bmatrix} a \amp b \amp \mid \amp 1 \amp 0 \\ c \amp d \amp \mid \amp 0 \amp 1 \end{bmatrix} \end{equation*}
    But it gets even better! Let’s do an example...
    🔗

    Example 2.2.10.

    Let’s say we want to find the inverse of the matrix \(A = \begin{bmatrix} 0 \amp 2 \\ 1 \amp -2 \end{bmatrix}\text{.}\) We will first make our big augmented matrix:
    \begin{equation*} \begin{bmatrix} 0 \amp 2 \amp \mid \amp 1 \amp 0 \\ 1 \amp -2 \amp \mid \amp 0 \amp 1 \end{bmatrix} \end{equation*}
    Now we will put this thing into reduced row echelon form!
    🔗
    Step 1: Swap Row 1 and Row 2, we get
    \begin{equation*} \begin{bmatrix} 1 \amp -2 \amp \mid \amp 0 \amp 1 \\ 0 \amp 2 \amp \mid \amp 1 \amp 0 \end{bmatrix} \end{equation*}
    We have our first leading \(1\) and it already has a zero beneath it.
    🔗
    Step 2: Multiply Row 2 by \(\frac{1}{2}\text{,}\) we get
    \begin{equation*} \begin{bmatrix} 1 \amp -2 \amp \mid \amp 0 \amp 1 \\ 0 \amp 1 \amp \mid \amp \frac{1}{2} \amp 0 \end{bmatrix} \end{equation*}
    Now the matrix is in row echelon form. Let’s get it into reduced row echelon form by getting a zero above the second leading \(1\text{.}\)
    🔗
    Step 3: Multiply \(2\) times Row 2 to Row 1 (and replace Row 1), we get
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp \mid \amp 1 \amp 1 \\ 0 \amp 1 \amp \mid \amp \frac{1}{2} \amp 0 \end{bmatrix} \end{equation*}
    Notice that the part on the left is the \(2 \times 2\) identity matrix. And, notice that the part on the right is the inverse!
    🔗
    🔗
    That is basically the technique:
    • Step 1: Make a big augmented matrix with \(A\) on the left side and \(I\) on the right side.
      🔗
      🔗
    • Step 2: Put the augmented matrix into reduced row echelon form.
      🔗
      🔗
    If the matrix is invertible, then the reduced row-echelon form will have \(I\) on the left side and \(A^{-1}\) on the right side.
    🔗

    Checkpoint 2.2.11.

    Use the Gaussian Elimination algorithm to find the inverse of the matrix
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 3 \\ 4 \amp 2 \amp -5 \\ 3 \amp 1 \amp -2 \end{bmatrix} \end{equation*}
    🔗
    Solution.
    The big augmented matrix is
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 3 \amp \mid \amp 1 \amp 0 \amp 0 \\ 4 \amp 2 \amp -5 \amp \mid \amp 0 \amp 1 \amp 0 \\ 3 \amp 1 \amp -2 \amp \mid \amp 0 \amp 0 \amp 1 \end{bmatrix} \end{equation*}
    🔗
    First let’s get zeros beneath our first leading \(1\text{:}\)
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 3 \amp \mid \amp 1 \amp 0 \amp 0 \\ 0 \amp 2 \amp -17 \amp \mid \amp -4 \amp 1 \amp 0 \\ 0 \amp 1 \amp -11 \amp \mid \amp -3 \amp 0 \amp 1 \end{bmatrix} \end{equation*}
    🔗
    Now we need to get our second leading \(1\) by multiplying Row 2 by \(\frac{1}{2}\text{:}\)
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 3 \amp \mid \amp 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp \frac{-17}{2} \amp \mid \amp -2 \amp \frac{1}{2} \amp 0 \\ 0 \amp 1 \amp -11 \amp \mid \amp -3 \amp 0 \amp 1 \end{bmatrix} \end{equation*}
    🔗
    Now we need a zero beneath our second leading \(1\text{.}\) We can multiply Row 2 by \(-1\) and adding it to Row 3:
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 3 \amp \mid \amp 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp \frac{-17}{2} \amp \mid \amp -2 \amp \frac{1}{2} \amp 0 \\ 0 \amp 0 \amp \frac{-5}{2} \amp \mid \amp -1 \amp \frac{-1}{2} \amp 1 \end{bmatrix} \end{equation*}
    🔗
    Now to get our last leading \(1\text{,}\) we need to multiply Row 3 by \(\frac{-2}{5}\text{:}\)
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 3 \amp \mid \amp 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp \frac{-17}{2} \amp \mid \amp -2 \amp \frac{1}{2} \amp 0 \\ 0 \amp 0 \amp 1 \amp \mid \amp \frac{2}{5} \amp \frac{1}{5} \amp \frac{-2}{5} \end{bmatrix} \end{equation*}
    🔗
    Then we need to get zeros above this third leading \(1\text{.}\) So, let’s add \(\frac{17}{2}\) times Row 3 to Row 2 and -3 times Row 3 to Row 1:
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 0 \amp \mid \amp \frac{-1}{5} \amp \frac{-3}{5} \amp \frac{6}{5} \\ 0 \amp 1 \amp 0 \amp \mid \amp \frac{7}{5} \amp \frac{11}{5} \amp \frac{-17}{5} \\ 0 \amp 0 \amp 1 \amp \mid \amp \frac{2}{5} \amp \frac{1}{5} \amp \frac{-2}{5} \end{bmatrix} \end{equation*}
    🔗
    So the inverse is
    \begin{equation*} \begin{bmatrix} \frac{-1}{5} \amp \frac{-3}{5} \amp \frac{6}{5} \\ \frac{7}{5} \amp \frac{11}{5} \amp \frac{-17}{5} \\ \frac{2}{5} \amp \frac{1}{5} \amp \frac{-2}{5} \end{bmatrix} \end{equation*}
    🔗
    🔗
    🔗
    🔗

    Subsection Important Theorems and Facts about Inverses

    Theorem 2.2.12. Cancellation Law for Matrices.

    Suppose \(B\) and \(C\) are matrices and \(A\) is an invertible matrix. Then the following holds:
    1. If \(AB = AC\text{,}\) then \(B = C\text{.}\)
      🔗
      🔗
    2. If \(BA = CA\text{,}\) then \(B = C\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.
    Part 1: Suppose \(AB = AC\text{.}\) Then
    \begin{align*} A^{-1} (AB) \amp= A^{-1} (AC) \\ \implies \qquad \lp A^{-1} A \rp B \amp= \lp A^{-1}A \rp C \\ \implies \qquad \qquad \quad \, \, IB \amp= IC \amp \end{align*}
    Therefore, \(B = C\)
    🔗
    The proof of Part 2 would be similar.
    🔗
    🔗

    Theorem 2.2.13.

    Suppose \(A\) is invertible. Then the inverse of \(A^T\) is the transpose of \(A^{-1}\text{.}\) That is,
    \begin{equation*} \lp A^T \rp^{-1} = \lp A^{-1} \rp^T \end{equation*}
    🔗
    🔗

    Proof.
    Suppose \(A\) is invertible. Then
    \begin{equation*} A^{-1}A = I \qquad \implies \qquad \lp A^{-1}A \rp^T = I^T \end{equation*}
    But \(I^T = I\) and \(\lp A^{-1} A \rp^T = A^T \lp A^{-1}\rp^T\text{.}\) Substituting these into the above equation, we get
    \begin{equation*} A^T \lp A^{-1} \rp^T = I \end{equation*}
    So the inverse of \(A^T\) is the transpose of \(A^{-1}\text{.}\)
    🔗
    🔗

    Theorem 2.2.14.

    If \(A\) and \(B\) are invertible matrices and \(AB\) is defined, then
    \begin{equation*} (AB)^{-1} = B^{-1}A^{-1} \end{equation*}
    🔗
    🔗
    This is also called "shoes and socks" because you take off your shoes and socks in the reverse order in which you put them on. That is what this theorem says about matrix inverses.
    🔗

    Proof.
    \begin{equation*} (AB)\lp B^{-1}A^{-1} \rp = A\lp BB^{-1} \rp A^{-1} = AIA^{-1} = AA^{-1} = I \end{equation*}
    🔗
    🔗
    There are also a collection of a bunch of facts about inverses (including some of the ones we have just mentioned).
    🔗
    Suppose the following matrices are all square and the same size.
    1. \(I\) is invertible and \(I^{-1} = I\text{.}\)
      🔗
      🔗
    2. If \(A\) is invertible, so is \(A^{-1}\) and \(\lp A^{-1} \rp^{-1} = A\text{.}\)
      🔗
      🔗
    3. If \(A\) and \(B\) are invertible, so is \(AB\text{,}\) and \((AB)^{-1} = B^{-1}A^{-1}\text{.}\)
      🔗
      🔗
    4. If \(A_1, A_2, \ldots, A_k\) are all invertible, so is their product \(A_1A_2\ldots A_k\text{,}\) and \(\lp A_1A_2\ldots A_k \rp^{-1} = A_k^{-1}\ldots A_2^{-1}A_1^{-1}\text{.}\)
      🔗
      🔗
    5. If \(A\) is invertible, then so is \(A^k\text{,}\) and \(\lp A^k \rp^{-1} = \lp A^{-1} \rp^k\text{.}\)
      🔗
      🔗
    6. If \(A\) is invertible and \(k \neq 0\) is a number, then \(kA\) is invertible and \(\lp kA \rp^{-1} = \frac{1}{k} A^{-1}\text{.}\)
      🔗
      🔗
    7. If \(A\) is invertible, so is its transpose \(A^T\text{,}\) and \(\lp A^T \rp^{-1} = \lp A^{-1} \rp^T\text{.}\)
      🔗
      🔗
    🔗

    Subsection Elementary Matrices

    There is an another way of finding the inverse of a matrix using elementary matrices. This is essentially the same as our Gaussian Elimination Algorithm but in another presentation using matrix multiplication.
    🔗
    We’ve learned how to solve systems of equations and find matrix inverses by using the Gaussian Elimination algorithm. And we learned how to multiply matrices. It turns out that these two things are related!
    🔗
    Each of the three elementary row operations can be performed by multiplying by an elementary matrix.
    🔗
    Let’s see this by looking at an example!
    🔗

    Example 2.2.15.

    Consider the system:
    \begin{equation*} \begin{cases} 3x + 2y = 5 \\ 5x - 8y = 14 \end{cases} \end{equation*}
    The augmented matrix is \(\begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 5 \amp -8 \amp \mid \amp 14 \end{bmatrix}\)
    🔗
    Recall there are three elementary row operations we can perform to an augmented matrix.
    1. We could swap the two rows by multiplying by the matrix \(\begin{bmatrix} 0 \amp 1 \\ 1 \amp 0 \end{bmatrix}\text{:}\)
      \begin{equation*} \begin{bmatrix} 0 \amp 1 \\ 1 \amp 0 \end{bmatrix} \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 5 \amp -8 \amp \mid \amp 14 \end{bmatrix} = \begin{bmatrix} 5 \amp -8 \amp \mid \amp 14 \\ 3 \amp 2 \amp \mid \amp 5 \end{bmatrix} \end{equation*}
      🔗
      🔗
    2. We could multiply Row 2 by \(5\) by multiplying by the matrix \(\begin{bmatrix} 1 \amp 0 \\ 0 \amp 5 \end{bmatrix}\text{:}\)
      \begin{equation*} \begin{bmatrix} 1 \amp 0 \\ 0 \amp 5 \end{bmatrix} \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 5 \amp -8 \amp \mid \amp 14 \end{bmatrix} = \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 25 \amp -40 \amp \mid \amp 70 \end{bmatrix} \end{equation*}
      🔗
      🔗
    3. We could add \(3\) times Row 1 to Row 2 (and replace Row 2) by multiplying by the matrix \(\begin{bmatrix} 1 \amp 0 \\ 3 \amp 1 \end{bmatrix}\text{:}\)
      \begin{equation*} \begin{bmatrix} 1 \amp 0 \\ 3 \amp 1 \end{bmatrix} \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 5 \amp -8 \amp \mid \amp 14 \end{bmatrix} = \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 14 \amp -2 \amp \mid \amp 29 \end{bmatrix} \end{equation*}
      🔗
      🔗
    Notice that in each case, the matrix that we used to do the elementary operation is exactly the matrix you would get by doing that same elementary operation to the identity matrix. WHAT!??
    1. If we swap the two rows of \(\begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix}\text{,}\) we get \(\begin{bmatrix} 0 \amp 1 \\ 1 \amp 0 \end{bmatrix}\)
      🔗
      🔗
    2. If we multiply Row 2 of \(\begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix}\) by \(5\text{,}\) we get \(\begin{bmatrix} 1 \amp 0 \\ 0 \amp 5 \end{bmatrix}\)
      🔗
      🔗
    3. If we add \(3\) times Row 1 of \(\begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix}\) to Row 2 (and replace Row 2), we get \(\begin{bmatrix} 1 \amp 0 \\ 3 \amp 1 \end{bmatrix}\)
      🔗
      🔗
    🔗
    🔗
    Summary: Every elementary row operation corresponds to an elementary matrix, \(E\text{,}\) that can be created by performing that operation on the identity matrix. We can perform that operation on any matrix \(A\) by computing the product \(EA\text{.}\)
    🔗

    Example 2.2.16.

    Consider the matrix \(A = \begin{bmatrix} 1 \amp 2 \\ 3 \amp 4 \end{bmatrix}\text{.}\) Suppose we did the following two steps:
    1. Swap Row 1 and Row 2;
      🔗
      🔗
    2. Add \(4\) times Row 1 to Row 2 (and replace Row 2).
      🔗
      🔗
    That would look like this:
    \begin{equation*} \begin{bmatrix} 1 \amp 2 \\ 3 \amp 4 \end{bmatrix} \rightarrow \begin{bmatrix} 3 \amp 4 \\ 1 \amp 2 \end{bmatrix} \rightarrow \begin{bmatrix} 3 \amp 4 \\ 13 \amp 18 \end{bmatrix} \end{equation*}
    🔗
    The first step corresponds to the elementary matrix
    \begin{equation*} E_1 = \begin{bmatrix} 0 \amp 1 \\ 1 \amp 0 \end{bmatrix} \end{equation*}
    and the second step corresponds to the elementary matrix
    \begin{equation*} E_2 = \begin{bmatrix} 1 \amp 0 \\ 4 \amp 1 \end{bmatrix} \end{equation*}
    (Note: When we made \(E_2\text{,}\) we started with a brand new identity matrix rather than using \(E_1\text{.}\))
    🔗
    Now notice that
    \begin{equation*} E_2E_1A = \begin{bmatrix} 1 \amp 0 \\ 4 \amp 1 \end{bmatrix} \begin{bmatrix} 0 \amp 1 \\ 1 \amp 0 \end{bmatrix} \begin{bmatrix} 1 \amp 2 \\ 3 \amp 4 \end{bmatrix} = \begin{bmatrix} 3 \amp 4 \\ 13 \amp 18 \end{bmatrix} \end{equation*}
    🔗
    🔗
    The inverse of an elementary matrix \(E\text{,}\) \(E^{-1}\) is also an elementary matrix and can be used to perform the inverse of the elementary operation (the operation that would undo the original row operation).
    🔗

    Example 2.2.17.

    If \(E\) corresponds to adding \(7\) times Row 1 to Row 3, then the elementary matrix is
    \begin{equation*} E = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 7 \amp 0 \amp 1 \end{bmatrix} \end{equation*}
    Then \(E^{-1} = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ -7 \amp 0 \amp 1 \end{bmatrix}\) and this corresponds to adding \(-7\) times Row 1 to Row 3.
    🔗
    🔗
    Here is the claim: A matrix is invertible if and only if it is the product of elementary matrices.
    🔗
    Let’s think about our method for finding inverses using Gaussian Elimination algorithm. We create a big augmented matrix \(\begin{bmatrix} A \mid I \end{bmatrix}\) and then do elementary operations until we get to \(\begin{bmatrix} I \mid A^{-1} \end{bmatrix}\text{.}\)
    🔗
    Now we know that we could do this by multiplying the augmented matrix by the corresponding elementary matrices. Let’s call those elementary matrices \(E_1, E_2, \ldots, E_k\text{.}\) Then if we just look at the right side of the big augmented matrix, we would see that
    \begin{equation*} E_k \ldots E_2E_1 I = A^{-1} \end{equation*}
    Take the inverse of both sides, and we get
    \begin{equation*} E_1^{-1} E_2^{-1} \ldots E_{k}^{-1} = A \end{equation*}
    🔗
    We can use what we just learned to describe a process that will always work to write any invertible matrix as a product of elementary matrices:
    1. Put the matrix into reduced row echelon form. If it is invertible, then this form will be the identity matrix.
      🔗
      🔗
    2. For each step, write down the corresponding elementary matrix (writing them from right to left). Remember that each time we need to start with a brand new identity matrix to make our elementary matrix!
      🔗
      If you multiply the elementary matrices, you will get the inverse of the original matrix.
      🔗
      🔗
    3. Take the inverse of each of the elementary matrices and then write them down in the reverse order. Then we are done!
      🔗
      🔗
    🔗

    Example 2.2.18.

    Consider this matrix \(A = \begin{bmatrix} 3 \amp 2 \\ 5 \amp -8 \end{bmatrix}\) and we want to find \(A^{-1}\text{.}\)
    🔗
    First, let’s multiply the top row by \(\frac{1}{3}\) to get our leading \(1\text{:}\)
    \begin{equation*} \begin{bmatrix} 1 \amp \frac{2}{3} \\ 5 \amp -8 \end{bmatrix} \end{equation*}
    This first step corresponds to the elementary matrix \(E_1 = \begin{bmatrix} \frac{1}{3} \amp 0 \\ 0 \amp 1 \end{bmatrix}\)
    🔗
    Second, let’s add \(-5\) times Row 1 to Row 2 (and replace Row 2) to get a zero below the first leading \(1\text{:}\)
    \begin{equation*} \begin{bmatrix} 1 \amp \frac{2}{3} \\ 0 \amp \frac{-34}{3} \end{bmatrix} \end{equation*}
    This second step corresponds to the elementary matrix \(E_2 = \begin{bmatrix} 1 \amp 0 \\ -5 \amp 1 \end{bmatrix}\)
    🔗
    Third, let’s multiply the second row by \(\frac{-3}{34}\) to get our second leading \(1\text{:}\)
    \begin{equation*} \begin{bmatrix} 1 \amp \frac{2}{3} \\ 0 \amp 1 \end{bmatrix} \end{equation*}
    This third step corresponds to the elementary matrix \(E_3 = \begin{bmatrix} 1 \amp 0 \\ 0 \amp \frac{-3}{34} \end{bmatrix}\)
    🔗
    Fourth, let’s add \(\frac{-2}{3}\) times Row 2 to Row 1 (and replace Row 1) to put the matrix into reduced row echelon form:
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \end{bmatrix} \end{equation*}
    This fourth step corresponds to the elementary matrix \(E_4 = \begin{bmatrix} 1 \amp \frac{-2}{3} \\ 0 \amp 1 \end{bmatrix}\)
    🔗
    So
    \begin{align*} A^{-1} \amp= E_4E_3E_2E_1 \\ \amp= \begin{bmatrix} 1 \amp \frac{-2}{3} \\ 0 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \\ 0 \amp \frac{-3}{34} \end{bmatrix} \begin{bmatrix} 1 \amp 0 \\ -5 \amp 1 \end{bmatrix} \begin{bmatrix} \frac{1}{3} \amp 0 \\ 0 \amp 1 \end{bmatrix} \\ \amp= \begin{bmatrix} \frac{4}{17} \amp \frac{1}{17} \\ \frac{5}{34} \amp \frac{-3}{34} \end{bmatrix} \end{align*}
    🔗
    We can write all of the inverses of the elementary matrices by using the reverse operation on the identity matrix:
    • The inverse of \(E_4 = \begin{bmatrix} 1 \amp \frac{-2}{3} \\ 0 \amp 1 \end{bmatrix}\) is \(\begin{bmatrix} 1 \amp \frac{2}{3} \\ 0 \amp 1 \end{bmatrix}\text{.}\)
      🔗
      🔗
    • The inverse of \(E_3 = \begin{bmatrix} 1 \amp 0 \\ 0 \amp \frac{-3}{34} \end{bmatrix}\) is \(\begin{bmatrix} 1 \amp 0 \\ 0 \amp \frac{34}{-3} \end{bmatrix}\text{.}\)
      🔗
      🔗
    • The inverse of \(E_2 = \begin{bmatrix} 1 \amp 0 \\ -5 \amp 1 \end{bmatrix}\) is \(\begin{bmatrix} 1 \amp 0 \\ 5 \amp 1 \end{bmatrix}\text{.}\)
      🔗
      🔗
    • The inverse of \(E_1 = \begin{bmatrix} \frac{1}{3} \amp 0 \\ 0 \amp 1 \end{bmatrix}\) is \(\begin{bmatrix} 3 \amp 0 \\ 0 \amp 1 \end{bmatrix}\text{.}\)
      🔗
      🔗
    So
    \begin{align*} A \amp= E_1^{-1}E_2^{-1}E_3^{-1}E_4^{-1} \\ \amp= \begin{bmatrix} 3 \amp 0 \\ 0 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \\ 5 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \\ 0 \amp \frac{34}{-3} \end{bmatrix} \begin{bmatrix} 1 \amp \frac{2}{3} \\ 0 \amp 1 \end{bmatrix} \end{align*}
    🔗
    🔗

    Checkpoint 2.2.19.

    Write the following matrix AND its inverse as a product of elementary matrices.
    \begin{equation*} A = \begin{bmatrix} 0 \amp 2 \amp 1 \\ 1 \amp 3 \amp -1 \\ -1 \amp -3 \amp 0 \end{bmatrix} \end{equation*}
    🔗
    Solution.
    Step 1: Swap Row 1 and Row 2 to get
    \begin{equation*} \begin{bmatrix} 1 \amp 3 \amp -1 \\ 0 \amp 2 \amp 1 \\ -1 \amp -3 \amp 0 \end{bmatrix} \end{equation*}
    This corresponds to the elementary matrix \(E_1 = \begin{bmatrix} 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\)
    🔗
    Step 2: Add \(1\) times Row 1 to Row 3 (and replace Row 3) to get
    \begin{equation*} \begin{bmatrix} 1 \amp 3 \amp -1 \\ 0 \amp 2 \amp 1 \\ 0 \amp 0 \amp -1 \end{bmatrix} \end{equation*}
    This corresponds to the elementary matrix \(E_2 = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 1 \end{bmatrix}\)
    🔗
    Step 3: Multiply Row 2 by \(\frac{1}{2}\) to get
    \begin{equation*} \begin{bmatrix} 1 \amp 3 \amp -1 \\ 0 \amp 1 \amp \frac{1}{2} \\ 0 \amp 0 \amp -1 \end{bmatrix} \end{equation*}
    This corresponds to the elementary matrix \(E_3 = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp \frac{1}{2} \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\)
    🔗
    Step 4: Multiply Row 3 by \(-1\) to get
    \begin{equation*} \begin{bmatrix} 1 \amp 3 \amp -1 \\ 0 \amp 1 \amp \frac{1}{2} \\ 0 \amp 0 \amp 1 \end{bmatrix} \end{equation*}
    This corresponds to the elementary matrix \(E_4 = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp -1 \end{bmatrix}\)
    🔗
    Step 5: Add \(-3\) times Row 2 to Row 1 (and replace Row 1) to get
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp -\frac{5}{2} \\ 0 \amp 1 \amp \frac{1}{2} \\ 0 \amp 0 \amp 1 \end{bmatrix} \end{equation*}
    This corresponds to the elementary matrix \(E_5 = \begin{bmatrix} 1 \amp -3 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\)
    🔗
    Step 6: Add \(\frac{5}{2}\) times Row 3 to Row 1 (and replace Row 1) to get
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp \frac{1}{2} \\ 0 \amp 0 \amp 1 \end{bmatrix} \end{equation*}
    This corresponds to the elementary matrix \(E_6 = \begin{bmatrix} 1 \amp 0 \amp \frac{5}{2} \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\)
    🔗
    Step 7: Add \(\frac{-1}{2}\) times Row 3 to Row 2 (and replace Row 2) to get
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \end{equation*}
    This corresponds to the elementary matrix \(E_7 = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp \frac{-1}{2} \\ 0 \amp 0 \amp 1 \end{bmatrix}\)
    🔗
    This tells us that
    \begin{align*} A^{-1} \amp= E_7E_6E_5E_4E_3E_2E_1 \\ \amp= \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp \frac{-1}{2} \\ 0 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \amp \frac{5}{2} \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp -3 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp -1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp \frac{1}{2} \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \end{align*}
    If we multiply those out we see that \(A^{-1} = \begin{bmatrix} \frac{-3}{2} \amp \frac{-3}{2} \amp \frac{-5}{2} \\ \frac{1}{2} \amp \frac{1}{2} \amp \frac{1}{2} \\ 0 \amp -1 \amp -1 \end{bmatrix}\)
    🔗
    And taking the inverse of both sides of the giant product (and using the "shoes and socks" rule), we see that
    \begin{equation*} A = E_1^{-1}E_2^{-1}E_3^{-1}E_4^{-1}E_5^{-1}E_6^{-1}E_7^{-1} \end{equation*}
    So let’s find the inverses of each of these matrices by performing the inverse operation on the identity matrix:
    • \(E_1\) swaps Row 1 and Row 2 and that operation is its own inverse (do it twice and you get right back where you started), so \(E_1^{-1} = \begin{bmatrix} 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\text{.}\)
      🔗
      🔗
    • \(E_2\) adds Row 1 to Row 3, and the inverse of that is adding \(-1\) times Row 1 to Row 2, so \(E_2^{-1} = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ -1 \amp 0 \amp 1 \end{bmatrix}\text{.}\)
      🔗
      🔗
    • \(E_3\) multiplies Row 2 by \(\frac{1}{2}\text{,}\) and the inverse of that is multiplying Row 2 by \(2\text{,}\) so \(E_3^{-1} = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 2 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\text{.}\)
      🔗
      🔗
    • \(E_4\) multiplies Row 3 by \(-1\text{,}\) and that operation is its own inverse, so \(E_4^{-1} = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp -1 \end{bmatrix}\text{.}\)
      🔗
      🔗
    • \(E_5\) adds \(-3\) times Row 2 to Row 1, and the inverse of that is adding \(3\) times Row 2 to Row 1, so \(E_5^{-1} = \begin{bmatrix} 1 \amp 3 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\text{.}\)
      🔗
      🔗
    • \(E_6\) adds \(\frac{5}{2}\) times Row 3 to Row 1, and the inverse of that is adding \(\frac{-5}{2}\) times Row 3 to Row 1, so \(E_6^{-1} = \begin{bmatrix} 1 \amp 0 \amp \frac{-5}{2} \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\text{.}\)
      🔗
      🔗
    • \(E_7\) adds \(\frac{-1}{2}\) times Row 3 to Row 2, and the inverse of that is adding \(\frac{1}{2}\) times Row 3 to Row 2, so \(E_7^{-1} = \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp \frac{1}{2} \\ 0 \amp 0 \amp 1 \end{bmatrix}\text{.}\)
      🔗
      🔗
    This means...
    \begin{align*} A \amp= E_1^{-1}E_2^{-1}E_3^{-1}E_4^{-1}E_5^{-1}E_6^{-1}E_7^{-1} \\ \amp= \begin{bmatrix} 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ -1 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 2 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp -1 \end{bmatrix} \begin{bmatrix} 1 \amp 3 \amp 0 \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \amp \frac{-5}{2} \\ 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix} \begin{bmatrix} 1 \amp 0 \amp 0 \\ 0 \amp 1 \amp \frac{1}{2} \\ 0 \amp 0 \amp 1 \end{bmatrix} \end{align*}
