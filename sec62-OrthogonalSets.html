    Section 5.2 Orthogonal Sets

    Recall back in the last section (and the next section), we saw that two vectors are orthogonal if their inner product is \(0\text{.}\) Similarly, we can define orthogonal sets as a set of orthogonal vectors.
    🔗

    Definition 5.2.1. Orthogonal Sets.

    A set of vectors \(\set{\v{v}_1, \v{v}_2 , \ldots, \v{v}_p}\) is an orthogonal set if each pair of distinct vectors from this set is orthogonal. That is,
    \begin{equation*} \v{v}_i \cdot \v{v}_j = 0 \end{equation*}
    whenever \(i \neq j\text{.}\)
    🔗
    🔗

    Example 5.2.2.

    Let \(\v{v}_1 = \begin{bmatrix} 3 \\ 1 \\ 1 \end{bmatrix}\text{,}\) \(\v{v}_2 = \begin{bmatrix} -1 \\ 2 \\ 1 \end{bmatrix}\text{,}\) and \(\v{v}_3 = \begin{bmatrix} -\frac{1}{2} \\ -2 \\ \frac{7}{2} \end{bmatrix}\text{.}\) Is this set \(\set{\v{v}_1, \v{v}_2, \v{v}_3}\) an orthogonal set?
    🔗
    If this is an orthogonal set, then each pair of the distinct vectors is orthogonal. We can form three pairs of distinct vectors in the set, and we will check their orthogonality respectively:
    \begin{align*} \v{v}_1 \cdot \v{v}_2 \amp= \begin{bmatrix} 3 \\ 1 \\ 1 \end{bmatrix} \cdot \begin{bmatrix} -1 \\ 2 \\ 1 \end{bmatrix} = -3 + 2 + 1 = 0\\ \v{v}_2 \cdot \v{v}_3 \amp= \begin{bmatrix} -1 \\ 2 \\ 1 \end{bmatrix} \cdot \begin{bmatrix} -\frac{1}{2} \\ -2 \\ \frac{7}{2} \end{bmatrix} = \frac{1}{2} - 4 + \frac{7}{2} = 0\\ \v{v}_1 \cdot \v{v}_3 \amp= \begin{bmatrix} 3 \\ 1 \\ 1 \end{bmatrix} \cdot \begin{bmatrix} -\frac{1}{2} \\ -2 \\ \frac{7}{2} \end{bmatrix} = -\frac{3}{2} - 2 + \frac{7}{2} = 0 \end{align*}
    Observe that each pair of distinct vectors in the set is orthogonal. Hence this is an orthogonal set.
    🔗
    🔗
    Sometimes we want to only focus on the direction of a vector. Since the length of a vector has nothing to do with its direction, we want to minimize the effect of the length by making the length to be \(1\).
    🔗

    Definition 5.2.3. Unit Vectors.

    A vector with length \(1\) is called a unit vector.
    🔗
    🔗
    Given a non-zero vector \(\v{v}\text{,}\) we can make its length to be \(1\) by dividing it by its length. That is, a unit vector in the same direction as \(\v{v}\) is
    \begin{equation*} \v{u} = \frac{\v{v}}{\|\v{v}\|} \end{equation*}
    This process of creating a normal vector \(\v{u}\) from \(\v{v}\) is called normalizing \(\v{v}\).
    🔗
    Given an orthogonal set, we can even normalize each vector in this set! This set is called the orthonormal set.
    🔗

    Definition 5.2.4. Orthonormal Sets.

    A set \(\set{\v{u}_1, \v{u}_2, \ldots, \v{u}_p}\) is an orthonormal set if it is an orthogonal set of unit vectors.
    🔗
    🔗

    Example 5.2.5.

    We showed that, in the previous example, that \(\set{\v{v}_1, \v{v}_2, \v{v}_3}\) is an orthogonal set, where \(\v{v}_1 = \begin{bmatrix} 3 \\ 1 \\ 1 \end{bmatrix}\text{,}\) \(\v{v}_2 = \begin{bmatrix} -1 \\ 2 \\ 1 \end{bmatrix}\text{,}\) and \(\v{v}_3 = \begin{bmatrix} -\frac{1}{2} \\ -2 \\ \frac{7}{2} \end{bmatrix}\text{.}\) Is this set also an orthonormal set?
    🔗
    The answer is no since none of the three vectors have the length of \(1\text{.}\)
    🔗
    But we can make an orthonormal set out of it by normalizing each vector. Since normalization (aka rescaling the vector to have the length of \(1\)) will not change the direction, the orthogonality remains.
    🔗
    Observe that
    \begin{align*} \|\v{v}_1\| \amp= \sqrt{3^2 + 1^2 + 1^2} = \sqrt{11} \\ \|\v{v}_2\| \amp= \sqrt{(-1)^2 + 2^2 + 1^2} = \sqrt{6} \\ \|\v{v}_3\| \amp= \sqrt{\lp -\frac{1}{2} \rp^2 + (-2)^2 + \lp \frac{7}{2} \rp^2} = \frac{\sqrt{66}}{2} \end{align*}
    Then we can normalize each vector by dividing it by its lengths:
    \begin{align*} \v{u}_1 \amp= \frac{\v{v}_1}{\|\v{v}_1\|} = \frac{1}{\sqrt{11}}\begin{bmatrix} 3 \\ 1 \\ 1 \end{bmatrix} = \begin{bmatrix} \frac{3}{\sqrt{11}} \\ \frac{1}{\sqrt{11}} \\ \frac{1}{\sqrt{11}} \end{bmatrix}\\ \v{u}_2 \amp= \frac{\v{v}_2}{\|\v{v}_2\|} = \frac{1}{\sqrt{6}}\begin{bmatrix} -1 \\ 2 \\ 1 \end{bmatrix} = \begin{bmatrix} -\frac{1}{\sqrt{6}} \\ \frac{2}{\sqrt{6}} \\ \frac{1}{\sqrt{6}} \end{bmatrix}\\ \v{u}_3 \amp= \frac{\v{v}_3}{\|\v{v}_3\|} = \frac{2}{\sqrt{66}}\begin{bmatrix} -\frac{1}{2} \\ -2 \\ \frac{7}{2} \end{bmatrix} = \begin{bmatrix} -\frac{1}{\sqrt{66}} \\ -\frac{4}{\sqrt{66}} \\ \frac{7}{\sqrt{66}} \end{bmatrix} \end{align*}
    The set \(\set{\v{u}_1, \v{u}_2, \v{u}_3}\) is now an orthonormal set!
    🔗
    🔗
    But why do we want orthogonal/orthonormal sets? That is because these sets are linearly independent so they can become some bases for some subspaces. Also, things are working out nicely with orthogonal/orthonormal bases, so we like to work with them.
    🔗
    If we want an orthogonal basis for a subspace, we can produce it using the Gram-Schmidt Process. We will not be covering it in this class. Feel free to look into it if you are interested. You should be able to understand the process just fine, as the trick to make the process work is through projection that we learned before!
