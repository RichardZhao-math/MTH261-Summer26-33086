    Section 1.5 Solution Sets of Linear Systems

    Now we get to the cool part! We are going to learn how to multiply a matrix by a vector. This is SUPER IMPORTANT for three reasons:
    1. This corresponds to evaluating a linear transformation at a point which leads to tons of cool geometry stuff. Much more about this in the future!
      🔗
      🔗
    2. It is the foundation for multiplying matrices (which is probably the most important thing in Matrix Algebra and corresponds to composing linear transformations).
      🔗
      🔗
    3. It gives us new ways to think about systems of equations!
      🔗
      🔗
    Takeaway: make sure you are an expert at multiplying a vector by a matrix!
    🔗

    Objectives

    After this section, students will be able to:
    🔗
    • represent a system of linear equations in four equivalent forms: a system of equations, an augmented matrix, a matrix equation, and a vector equation.
      🔗
      🔗
    • multiply a matrix by a vector using both the dot product method and the linear combination method.
      🔗
      🔗
    • determine the existence of solutions for a system of linear equations by comparing the rank of the coefficient matrix and the rank of the augmented matrix.
      🔗
      🔗
    • see the connection between matrix-vector multiplication and linear transformations.
      🔗
      🔗
    • identify and apply the properties of two special matrices: the identity matrix and the zero matrix, in the context of multiplying a matrix by a vector.
      🔗
      🔗
    🔗
    There are two types of systems of linear equations if we consider the values of the constants terms: homogeneous system and non-homogeneous system. They are of course related and we will focus on the homogeneous systems first, then we will see how to use the results to understand the non-homogeneous systems.
    🔗

    Subsection Homogeneous Linear System

    Let’s start by looking at a quick task! Richard claims that the following system of linear equations has at least one solution (hence a consistent system) at the first glance.
    \begin{equation*} \begin{cases} 4x - 12y + 2z = 0 \\ 2x - 6y = 0 \\ 3x - 9y + z = 0 \\ -2x + 6y - z = 0 \end{cases} \end{equation*}
    Why does Richard know about it quickly?
    🔗
    Recall something is called a solution if it satisfies all the equations in the system! Richard made a keen observation that all equations would work out if the variables are all zero. That is, \(x = 0\text{,}\) \(y = 0\text{,}\) and \(z = 0\text{.}\)
    🔗
    To verify, Richard plugged in zero for every variable. He got zero on the left-hand side of every equation and solve all of them! So \((0,0,0)\) is a solution to this system of linear equations.
    🔗
    It turns out that this type of systems of linear equations is called a homogeneous system!
    🔗

    Definition 1.5.1.

    A homogeneous system of equations (or a system of homogeneous equations) is a system of linear equations where all of the constants to the right of the equal sign are zero.
    🔗
    🔗
    For example, we worked with the following system of linear equations in a previous section:
    \begin{equation*} \begin{cases} 4x - 12y + 2z = -12 \\ 2x - 6y = -10 \\ 3x - 9y + z = -11 \\ -2x + 6y - z = 6 \end{cases} \end{equation*}
    This is NOT a homogeneous system since the constants on the right-hand side are not all zero. We can make it homogeneous by changing the number on the right-hand side of every equation to a zero as follows:
    \begin{equation*} \begin{cases} 4x - 12y + 2z = 0 \\ 2x - 6y = 0 \\ 3x - 9y + z = 0 \\ -2x + 6y - z = 0 \end{cases} \end{equation*}
    Then we have a homogeneous system!
    🔗
    Note: Just to be clear, this is a different system of linear equations. However, we will see later that it is actually related to the original one in an important way later on!
    🔗
    Usually, a system of linear equations can have either (1) zero, (2) one, or (3) infinitely many solutions. But it is impossible for a homogeneous system to have zero solutions! As we figured out the very first question in this section, every homogeneous system has at least the trivial solution (where all the variables are zero). So a homogeneous system either has one solution (trivial) OR it has infinitely many solutions.
    🔗
    Let’s solve our homogeneous system! First, we will write the augmented matrix for the system:
    \begin{equation*} \begin{bmatrix} 4 \amp -12 \amp 2 \amp \mid \amp 0 \\ 2 \amp -6 \amp 0 \amp \mid \amp 0 \\ 3 \amp -9 \amp 1 \amp \mid \amp 0 \\ -2 \amp 6 \amp -1 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    We can actually do the exact same steps that we did with the non-homogeneous version. The only difference is that the constant column doesn’t require any work since all the entries hold steady at zero.
    🔗
    So if we put it into row echelon form, we get:
    \begin{equation*} \begin{bmatrix} 1 \amp -3 \amp \frac{1}{2} \amp \mid \amp 0 \\ 0 \amp 0 \amp 1 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    Now we can convert back to a system of linear equations and we get:
    \begin{equation*} \begin{cases} x - 3y + \frac{1}{2}z = 0 \\ z = 0 \\ 0 = 0 \\ 0 = 0 \end{cases} \end{equation*}
    Just like with the regular version, \(x\) and \(z\) are the leading variables and \(y\) is a free variable. We can back substitute \(z = 0\) into the first equation to get \(x = 3y\) and write the solution in parametric form:
    \begin{equation*} \begin{cases} x = 3y \\ y = y \\ z = 0 \end{cases} \end{equation*}
    or, using our linear algebra notation of vectors (which means in column vectors), an arbitrary vector in the solution set looks like: \(\begin{bmatrix} 3y \\ y \\ 0 \end{bmatrix}\text{.}\)
    🔗
    Now, notice that we can factor out a \(y\) in the solution and see that the set of solutions is of the form \(y\begin{bmatrix} 3 \\ 1 \\ 0 \end{bmatrix}\) where \(y\) can be any real number. That means every solution of the system is a multiple of \(\begin{bmatrix} 3 \\ 1 \\ 0 \end{bmatrix}\text{.}\) Alternatively, we can say that every solution is in \(\span\left\{\begin{bmatrix} 3 \\ 1 \\ 0 \end{bmatrix}\right\}\text{.}\)
    🔗
    We call this \(\begin{bmatrix} 3 \\ 1 \\ 0 \end{bmatrix}\) a "basis" or "basic vector" of the solution set (Actually "basis" is much more common, but some textbooks use "basic"...).
    🔗

    Example 1.5.2.

    Pretend that we had a (homogeneous) system of 4 linear equations in 5 variables \((x,y,z,u,z)\) and we put the augmented matrix into row echelon form and got:
    \begin{equation*} \begin{bmatrix} 1 \amp -3 \amp 5 \amp 2 \amp 7 \amp \mid \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \amp 1 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    The leading variables are \(x\text{,}\) \(z\text{,}\) and \(v\text{,}\) so our free variables are \(y\) and \(u\text{.}\) That is, we should be able to solve for the other three variables in terms of \(y\) and \(u\text{.}\)
    🔗
    Let’s convert it to a system of linear equations:
    \begin{equation*} \begin{cases} x - 3y + 5z + 2u + 7v = 0 \\ z + v = 0 \\ v = 0 \\ 0 = 0 \end{cases} \end{equation*}
    We know that \(v = 0\) and we can back substitute into the second equation to get \(z = 0\) as well.
    🔗
    Back substituting into the first equation, we get
    \begin{equation*} x - 3y + 2u = 0 \qquad \implies \qquad x = 3y - 2u \end{equation*}
    Now we can write an arbitrary point in the solution set as
    \begin{equation*} \begin{bmatrix} 3y - 2u \\ y \\ 0 \\ u \\ 0 \end{bmatrix} \end{equation*}
    Now we can split this up into two vectors, one with only \(y\)’s and one with only \(u\)’s:
    \begin{align*} \begin{bmatrix} 3y - 2u \\ y \\ 0 \\ u \\ 0 \end{bmatrix} \amp= \begin{bmatrix} 3y \\ y \\ 0 \\ 0 \\ 0 \end{bmatrix} + \begin{bmatrix} -2u \\ 0 \\ 0 \\ u \\ 0 \end{bmatrix} \\ \amp= y \begin{bmatrix} 3 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix} + u \begin{bmatrix} -2 \\ 0 \\ 0 \\ 1 \\ 0 \end{bmatrix} \amp\amp \text{factor out the $y$ and $u$} \end{align*}
    This means that every solution is a linear combination of the "basic" solutions \(\begin{bmatrix} 3 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix}\) and \(\begin{bmatrix} -2 \\ 0 \\ 0 \\ 1 \\ 0 \end{bmatrix}\text{.}\) Hence, the solution set is \(\span\left\{\begin{bmatrix} 3 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix}, \begin{bmatrix} -2 \\ 0 \\ 0 \\ 1 \\ 0 \end{bmatrix}\right\}\text{.}\)
    🔗
    Just to be clear, the two basis vectors \(\begin{bmatrix} 3 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix}\) and \(\begin{bmatrix} -2 \\ 0 \\ 0 \\ 1 \\ 0 \end{bmatrix}\) are solutions as well! That is one reason why they called the basic solutions (it explains the "solution" part). You can also imagine that we can construct infinitely many solutions using these two vectors as building blocks, so they are the "basis vectors" for the solution set!
    🔗
    🔗

    Checkpoint 1.5.3.

    Solve the following homogeneous system of linear equations and write an arbitrary solution as a linear combination of basic solutions (written as column vectors):
    \begin{equation*} \begin{cases} 3x - 2y + z + 5u = 0 \\ x - 6y + 2z + u = 0 \\ x + 10y - 3z + 3u = 0 \\ 4x - 8y + 3z + 6u = 0 \end{cases} \end{equation*}
    🔗
    Solution.
    Let’s start by writing the augmented matrix for the system:
    \begin{equation*} \begin{bmatrix} 3 \amp -2 \amp 1 \amp 5 \amp \mid \amp 0 \\ 1 \amp -6 \amp 2 \amp 1 \amp \mid \amp 0 \\ 1 \amp 10 \amp -3 \amp 3 \amp \mid \amp 0 \\ 4 \amp -8 \amp 3 \amp 6 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    We will solve the system using the Gaussian elimination algorithm.
    🔗
    Step 1: Swap Row 1 and Row 2 so that we get a leading \(1\) without using any fractions:
    \begin{equation*} \begin{bmatrix} 1 \amp -6 \amp 2 \amp 1 \amp \mid \amp 0 \\ 3 \amp -2 \amp 1 \amp 5 \amp \mid \amp 0 \\ 1 \amp 10 \amp -3 \amp 3 \amp \mid \amp 0 \\ 4 \amp -8 \amp 3 \amp 6 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    🔗
    Step 2: Add \(-3\) times Row 1 to Row 2 (and replace Row 2), add \(-1\) times Row 1 to Row 3 (and replace Row 3), and add \(-4\) times Row 1 to Row 4 (and replace Row 4) to get:
    \begin{equation*} \begin{bmatrix} 1 \amp -6 \amp 2 \amp 1 \amp \mid \amp 0 \\ 0 \amp 16 \amp -5 \amp 2 \amp \mid \amp 0 \\ 0 \amp 16 \amp -5 \amp 2 \amp \mid \amp 0 \\ 0 \amp 16 \amp -5 \amp 2 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    🔗
    Step 3: Multiply Row 2, Row 3, and Row 4 by \(\frac{1}{16}\) (observe the last three rows are exactly the same!) to get:
    \begin{equation*} \begin{bmatrix} 1 \amp -6 \amp 2 \amp 1 \amp \mid \amp 0 \\ 0 \amp 1 \amp -\frac{5}{16} \amp \frac{1}{8} \amp \mid \amp 0 \\ 0 \amp 1 \amp -\frac{5}{16} \amp \frac{1}{8} \amp \mid \amp 0 \\ 0 \amp 1 \amp -\frac{5}{16} \amp \frac{1}{8} \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    🔗
    Step 4: Add \(-1\) times Row 2 to Row 3 (and replace Row 3) and add \(-1\) times Row 2 to Row 4 (and replace Row 4) to get:
    \begin{equation*} \begin{bmatrix} 1 \amp -6 \amp 2 \amp 1 \amp \mid \amp 0 \\ 0 \amp 1 \amp -\frac{5}{16} \amp \frac{1}{8} \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    🔗
    We can see from the leading \(1\)s that \(x\) and \(y\) are the leading variables and \(z\) and \(u\) are the free variables. We can also see this from converting back to a system of linear equations:
    \begin{equation*} \begin{cases} x - 6y + 2z + u = 0 \\ y - \frac{5}{16}z + \frac{1}{8}u = 0 \\ 0 = 0 \\ 0 = 0 \end{cases} \end{equation*}
    We can solve for \(x\) in terms of \(y\text{,}\) \(z\text{,}\) and \(u\) to get:
    \begin{equation*} x = 6y - 2z - u \end{equation*}
    We can also solve for \(y\) in terms of \(z\) and \(u\) to get:
    \begin{equation*} y = \frac{5}{16}z - \frac{1}{8}u \end{equation*}
    Then we can back substitute this expression for \(y\) into the first equation to get:
    \begin{align*} x \amp= 6 \lp \frac{5}{16}z - \frac{1}{8}u \rp - 2z - u \\ \amp- \frac{15}{8}z - \frac{3}{4}u - 2z - u \\ \amp= -\frac{1}{8}z - \frac{7}{4}u \end{align*}
    Now let’s write our general solution as a column vector! Every solution looks like
    \begin{align*} \begin{bmatrix} -\frac{1}{8}z - \frac{7}{4}u \\ \frac{5}{16}z - \frac{1}{8}u \\ z \\ u \end{bmatrix} \amp= \begin{bmatrix} -\frac{1}{8}z \\ \frac{5}{16}z \\ z \\ 0 \end{bmatrix} + \begin{bmatrix} -\frac{7}{4}u \\ -\frac{1}{8}u \\ 0 \\ u \end{bmatrix} \\ \amp= z \begin{bmatrix} -\frac{1}{8} \\ \frac{5}{16} \\ 1 \\ 0 \end{bmatrix} + u \begin{bmatrix} -\frac{7}{4} \\ -\frac{1}{8} \\ 0 \\ 1 \end{bmatrix} \end{align*}
    🔗
    Note 1.5.4. If you want to turn the augmented matrix into reduced row echelon form....
    To get the augmented matrix into reduced row echelon form, we will just need to do one more step after Step 4.
    🔗
    Step 5: Add \(6\) times Row 2 to Row 1 (and replace Row 1) to get:
    \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp \frac{1}{8} \amp \frac{7}{4} \amp \mid \amp 0 \\ 0 \amp 1 \amp -\frac{5}{16} \amp \frac{1}{8} \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp \mid \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    🔗
    Converting it back to the system of linear equations, we get:
    \begin{equation*} \begin{cases} x + \frac{1}{8}z + \frac{7}{4}u = 0 \\ y - \frac{5}{16}z + \frac{1}{8}u = 0 \\ 0 = 0 \\ 0 = 0 \end{cases} \end{equation*}
    Similarly, we see that \(z\) and \(u\) are the free variables, and hence we will represent \(x\) and \(y\) in terms of \(z\) and \(u\text{:}\)
    \begin{equation*} \begin{cases} x = -\frac{1}{8}z - \frac{7}{4}u \\ y = \frac{5}{16}z - \frac{1}{8}u \end{cases} \end{equation*}
    🔗
    Then we can write our general solution as a column vector! Every solution looks like
    \begin{equation*} \begin{bmatrix} -\frac{1}{8}z - \frac{7}{4}u \\ \frac{5}{16}z - \frac{1}{8}u \\ z \\ u \end{bmatrix} = z \begin{bmatrix} -\frac{1}{8} \\ \frac{5}{16} \\ 1 \\ 0 \end{bmatrix} + u \begin{bmatrix} -\frac{7}{4} \\ -\frac{1}{8} \\ 0 \\ 1 \end{bmatrix} \end{equation*}
    🔗
    🔗
    🔗
    🔗
    Terminology! The way we write an arbitrary solution as a linear combination of basis vectors is called the parametric form of the solution set. If you are instructed to write the solution set in parametric form, you should write it as a linear combination of basis vectors using free variables as parameters.
    🔗

    Note 1.5.5. But Richard... Are there only one set of basis vectors for a solution set?

    The short answer is NO! There are many different ways to represent the basis vectors for a solution set. Sometimes we pick "nice" basis vectors to make the answer look pretty.
    🔗
    For example, in the checkpoint problem earlier, the solution in parametric form is
    \begin{equation*} z \begin{bmatrix} -\frac{1}{8} \\ \frac{5}{16} \\ 1 \\ 0 \end{bmatrix} + u \begin{bmatrix} -\frac{7}{4} \\ -\frac{1}{8} \\ 0 \\ 1 \end{bmatrix} \end{equation*}
    But the two basis vectors look messy because of the fractions.
    🔗
    We can actually represent these two basis vectors differently. If \(z = 16\text{,}\) then the first basis vector becomes
    \begin{equation*} 16 \begin{bmatrix} -\frac{1}{8} \\ \frac{5}{16} \\ 1 \\ 0 \end{bmatrix} = \begin{bmatrix} -2 \\ 5 \\ 16 \\ 0 \end{bmatrix} \end{equation*}
    If \(u = 8\text{,}\) then the second basis vector becomes
    \begin{equation*} 8 \begin{bmatrix} -\frac{7}{4} \\ -\frac{1}{8} \\ 0 \\ 1 \end{bmatrix} = \begin{bmatrix} -14 \\ -1 \\ 0 \\ 8 \end{bmatrix} \end{equation*}
    So we can also write the solution, in parametric form, as
    \begin{equation*} z \begin{bmatrix} -2 \\ 5 \\ 16 \\ 0 \end{bmatrix} + u \begin{bmatrix} -14 \\ -1 \\ 0 \\ 8 \end{bmatrix} \end{equation*}
    Look how much nicer this representation of the solution set is!
    🔗
    In the next section, we will talk about a concept called "independence". Basically, the idea here is that we swap two messy linearly independent basis vectors for two nice linearly independent vectors that span the same solution set.
    🔗
    🔗
    Recall the solution set to a homogeneous system can be represented as the span of the basis vectors. We know how spans look like geometrically!
    • If there are no free variables, then the only solution is the trivial solution, and \(\span\left\{\v{0}\right\} = \left\{\v{0}\right\}\) (just a point).
      🔗
      🔗
    • If there is one free variable, then the solution set is \(\span\left\{\v{v}_1\right\}\text{.}\) We saw in the last section that this will be a line through the origin (since \(\v{0}\) must be in the span).
      🔗
      🔗
    • If there are two free variables, then the solution set is \(\span\left\{\v{v}_1, \v{v}_2\right\}\text{.}\) We saw in the last section that this will be a plane through the origin (again, since \(\v{0}\) must be in the span).
      🔗
      🔗
    And the list goes on if we have more free variables!
    🔗
    🔗

    Subsection Non-Homogeneous Linear System

    By contrast, a non-homogeneous system of equations (or a system of non-homogeneous equations) is a system of linear equations where at least one of the constants to the right of the equal sign is NOT zero (so NOT a homogeneous system).
    🔗
    There is a very important relationship between the solution set of a non-homogeneous system and the solution set of the corresponding homogeneous system! Let’s revisit a non-homogeneous system we worked with before:
    \begin{equation*} \begin{cases} 4x - 12y + 2z = -12 \\ 2x - 6y = -10 \\ 3x - 9y + z = -11 \\ -2x + 6y - z = 6 \end{cases} \end{equation*}
    Skimming through Section 1.2 will help us locate the solution set of this system as
    \begin{equation*} x = 3y - 5\quad , \quad y = y \quad, \, \text{ and } \quad z = 4 \end{equation*}
    Or, putting it in column vector form, we can write an arbitrary solution as
    \begin{equation*} \begin{bmatrix} 3y - 5 \\ y \\ 4 \end{bmatrix} \end{equation*}
    Of course, we can do our previous trick of splitting this up into two vectors, one with only \(y\)’s and one without any \(y\)’s (so that we can factor out the \(y\)):
    \begin{equation*} \begin{bmatrix} 3y - 5 \\ y \\ 4 \end{bmatrix} = \begin{bmatrix} 3y \\ y \\ 0 \end{bmatrix} + \begin{bmatrix} -5 \\ 0 \\ 4 \end{bmatrix} = y \begin{bmatrix} 3 \\ 1 \\ 0 \end{bmatrix} + \begin{bmatrix} -5 \\ 0 \\ 4 \end{bmatrix} \end{equation*}
    Hmm... Does the first vector(s) \(y \begin{bmatrix} 3 \\ 1 \\ 0 \end{bmatrix}\) look familiar? This is actually the same as the solution set of the corresponding homogeneous system!
    🔗
    One can verify quickly that the second vector \(\begin{bmatrix} -5 \\ 0 \\ 4 \end{bmatrix}\text{,}\) by itself, is also a solution to the non-homogeneous system as well! This is our BIG theorem in this section!
    🔗

    Theorem 1.5.6.

    Suppose the equation \(A\v{x} = \v{b}\) is consistent for some given \(\v{b}\text{,}\) and let \(\v{p}\) be a solution to this equation. Then the solution set of \(A\v{x} = \v{b}\) is the set of all vectors of the form \(\v{w} = \v{p} + t\v{v}_h\text{,}\) where \(t\) is a parameter and \(\v{v}_h\) is a solution to the homogeneous equation \(A\v{x} = \v{0}\text{.}\)
    🔗
    🔗
    In general, we can solve a system of linear equations using the Gaussian Elimination algorithm. Assuming the system is consistent, we can write an arbitrary solution in parametric form using the same trick that we have been doing: split up the vector by the variable (or lackof) and factor out the free variable to get a linear combination of the basis vectors of the homogeneous solution set plus a particular solution to the non-homogeneous system.
    🔗

    Checkpoint 1.5.7.

    Describe all the solutions to the Matrix Equation \(A\v{x} = \v{b}\) where
    \begin{equation*} A = \begin{bmatrix} 1 \amp 3 \amp -5 \\ 1 \amp 4 \amp -8 \\ -3 \amp -7 \amp 9 \end{bmatrix} \qquad \text{ and }\qquad \v{b} = \begin{bmatrix} 4 \\ 7 \\ -6 \end{bmatrix} \end{equation*}
    in parametric form.
    🔗
    Answer.
    \(\v{x} = \begin{bmatrix} -5 \\ 3 \\ 0 \end{bmatrix} + t\begin{bmatrix} -4 \\ 3 \\ 1 \end{bmatrix}\)
    🔗
    🔗
    🔗
    We can also think about what it means in terms of geometry! Observe that the homogeneous equation \(A\v{x} = \v{0}\) looks a lot like our linear equation \(y = mx\) (in the sense that they are both linear and the constant term is zero). Then that means the non-homogeneous equation \(A\v{x} = \v{b}\) is like the linear equation \(y = mx + b\text{.}\)
    🔗
    How did we obtain \(y = mx + b\) from \(y = mx\text{?}\) The answer is vertical translation by \(b\text{!}\) That is, we can think of the solution to the non-homogeneous equation as the solution to the corresponding homogeneous equation shifted a bit by a particular solution to the non-homogeneous equation.
    🔗
    Geometrically, let’s think about \(\v{v}\) as a solution to \(A\v{x} = \v{0}\) and \(\v{p}\) as a particular solution to \(A\v{x} = \v{b}\text{.}\) Then we know that \(\v{p} + \v{v}\) is also a solution to \(A\v{x} = \v{b}\text{!}\) The diagram below shows the relationship between the two solutions to \(A\v{x} = \v{b}\text{:}\)
    🔗
    In addition, we can grab a vector from the span of \(\v{v}\) and add it to \(\v{p}\) to get another solution to \(A\v{x} = \v{b}\text{.}\) To keep things simple, let’s just imagine there is only one free variable in the homogeneous solution set. That is, the span of \(\v{v}\) is a line through the origin. Then the solution set of \(A\v{x} = \v{b}\) is a line parallel to the span of \(\v{v}\) that goes through the particular solution \(\v{p}\text{.}\)
    🔗
    Similarly, if there are two free variables in the homogeneous solution set, then the solution set of \(A\v{x} = \v{b}\) is a plane parallel to the span of the two basis vectors that goes through the particular solution \(\v{p}\text{.}\) And the list goes on if there are more free variables!
