    Section 1.8 The Matrix of a Linear Transformations

    So far, we have explored linear transformations by observing how they map inputs to outputs using the properties of linearity. While doing this step-by-step gives us a great feel for the mechanics, it would be highly efficient to capture the entire behavior of a transformation within a single, organized structure.
    🔗
    In this section, we will bridge the gap between abstract mappings and concrete computations. By noticing patterns in how transformations affect our simplest structural building blocks—the standard basis vectors—we will learn how to construct a standard matrix for any linear transformation. We will also use structural reasoning to translate geometric actions, like rotations and reflections, directly into matrix form, and determine if a transformation is an injection or surjection simply by analyzing the columns of its matrix.
    🔗

    Objectives

    • construct the standard matrix for a linear transformation.
      🔗
      🔗
    • connect geometric actions to their corresponding matrix representations.
      🔗
      🔗
    • determine if a linear transformation is injective (one-to-one) or surjective (onto).
      🔗
      🔗
    🔗

    Subsection To find the Standard Matrix...

    In the previous section, we left with an example where we find the image of the linear transformation using the linearity property without knowing the matrix. But it would be so nice to actually know the special matrix that defines the transformation. In this section, we will figure out a method to find this matrix.
    🔗
    Notice that any vector in \(\R^2\) can be written as a linear combination of \(\v{e}_1 = \begin{bmatrix} 1 \\ 0 \end{bmatrix}\) and \(\v{e}_2 = \begin{bmatrix} 0 \\ 1 \end{bmatrix}\text{.}\)
    🔗

    Quick proof of this claim.
    Let \(\begin{bmatrix} x \\ y \end{bmatrix}\) be an arbitrary vector in \(\R^2\text{.}\) Then
    \begin{equation*} \begin{bmatrix} x \\ y \end{bmatrix} = x\begin{bmatrix} 1 \\ 0 \end{bmatrix} + y\begin{bmatrix} 0 \\ 1 \end{bmatrix} = x\v{e}_1 + y\v{e}_2 \end{equation*}
    Proved!
    🔗
    🔗
    \(\left\{ \v{e}_1, \v{e}_2 \right\}\) is called the standard basis of \(\R^2\text{.}\)
    🔗
    For \(\R^n\text{,}\) the standard basis consists of the vectors \(\left\{\v{e}_1, \v{e}_2, \ldots, \v{e}_n\right\}\) where the \(k\)-th entry of \(\v{e}_k\) is a \(1\) and all of the rest are zeros. Clearly these are all \(n\)-dimensional vectors. That means the \(\v{e}_1\) is actually a different vector depending on which \(\R^n\) you are working with.
    🔗
    For example, \(\v{e}_1\) in \(\R^2\) is \(\begin{bmatrix} 1 \\ 0 \end{bmatrix}\text{,}\) but \(\v{e}_1\) in \(\R^3\) is \(\begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}\text{.}\) They are NOT the same vectors!
    🔗
    Combining this fact with the two properties that every linear transformation satisfies, we can see that every linear combination defined on \(\R^2\) (this works for \(\R^n\) as well, but let’s start small!) is actually determined by what it does to the standard basis vectors \(\v{e}_1\) and \(\v{e}_2\text{.}\)
    🔗

    Quick proof of this claim.
    Let \(\v{x}\) be a vector in \(\R^2\text{.}\) Then there exist scalars \(a, b \in \R\) such that
    \begin{equation*} \v{x} = a\v{e}_1 + b\v{e}_2 \end{equation*}
    Then for any linear transformation \(T\) defined on \(\R^2\text{,}\) we have
    \begin{equation*} T(\v{x}) = T(a\v{e}_1 + b\v{e}_2) = aT(\v{e}_1) + bT(\v{e}_2) \end{equation*}
    So the output of \(T\) at \(\v{x}\) is completely determined by the output of \(T\) at \(\v{e}_1\) and \(\v{e}_2\text{.}\)
    🔗
    🔗
    We can use this fact to figure out how to find the matrix for a linear transformation (and in doing so, we really prove that such a matrix will always exist for any linear transformation)!
    🔗

    Example 1.8.1.

    Suppose that \(T: \R^2 \to \R^3\) is a linear transformation such that
    \begin{equation*} T\lp \v{e}_1 \rp = \begin{bmatrix} 2 \\ 0 \\ 3 \end{bmatrix} \qquad \text{ and } \qquad T\lp \v{e}_2 \rp = \begin{bmatrix} 1 \\ 2 \\ 3 \end{bmatrix} \end{equation*}
    What is the matrix \(A\) such that \(T(\v{x}) = A\v{x}\) for all \(\v{x} \in \R^2\text{?}\)
    🔗
    Consider any vector \(\v{x} = \begin{bmatrix} x \\ y \end{bmatrix}\) in \(\R^2\text{.}\) We know that
    \begin{align*} T\lp \begin{bmatrix} x \\ y \end{bmatrix} \rp \amp= T\lp x\begin{bmatrix} 1 \\ 0 \end{bmatrix} + y\begin{bmatrix} 0 \\ 1 \end{bmatrix} \rp \\ \amp= xT\lp \begin{bmatrix} 1 \\ 0 \end{bmatrix} \rp + yT\lp \begin{bmatrix} 0 \\ 1 \end{bmatrix} \rp \\ \amp= x\begin{bmatrix} 2 \\ 0 \\ 3 \end{bmatrix} + y\begin{bmatrix} 1 \\ 2 \\ 3 \end{bmatrix} \\ \amp= \begin{bmatrix} 2x + y \\ 2y \\ 3x + 3y \end{bmatrix} \end{align*}
    Notice that the output is what we would get if we multiplied the matrix \(\begin{bmatrix} 2 \amp 1 \\ 0 \amp 2 \\ 3 \amp 3 \end{bmatrix}\) by the vector \(\begin{bmatrix} x \\ y \end{bmatrix}\text{!}\) So
    \begin{equation*} T\lp \begin{bmatrix} x \\ y \end{bmatrix} \rp = \begin{bmatrix} 2 \amp 1 \\ 0 \amp 2 \\ 3 \amp 3 \end{bmatrix} \begin{bmatrix} x \\ y \end{bmatrix} \end{equation*}
    We found the matrix that goes with this linear transformation!
    🔗
    🔗
    Notice that in the above example, the columns of our matrix were the outputs of the standard basis vectors. That tells us exactly how to find a matrix for any linear transformation!
    🔗
    Suppose \(T: \R^n \to \R^m\) is a linear transformation. To find the matrix \(A\) such that \(T\) is given by the rule \(T(\v{x}) = A\v{x}\text{,}\) we...
    1. Figure out \(T\lp \v{e_1} \rp, T\lp \v{e_2} \rp, \ldots T\lp \v{e_n} \rp\)
      🔗
      🔗
    2. Write the matrix that has columns \(T\lp \v{e_1} \rp, T\lp \v{e_2} \rp, \ldots T\lp \v{e_n} \rp\text{:}\)
      \begin{equation*} A = \begin{bmatrix} T\lp \v{e_1} \rp \amp T\lp \v{e_2} \rp \amp \cdots \amp T\lp \v{e_n} \rp \end{bmatrix} \end{equation*}
      🔗
      🔗
    3. Pat yourself on the back
      🔗
      🔗
    Special terminology! This matrix \(A\) that we found is called the standard matrix for the linear transformation \(T\text{.}\) We can find the standard matrix by figuring out where the transformation takes the standard basis vectors to.
    🔗

    Example 1.8.2.

    Let \(T: \R^2 \to \R^2\) be a linear transformation given by
    \begin{equation*} T\lp \begin{bmatrix} x \\ y \end{bmatrix} \rp = \begin{bmatrix} 2x + 3y \\ 5x + 7y \end{bmatrix} \end{equation*}
    Let’s find the matrix \(A\) such that \(T(\v{x}) = A\v{x}\) for all \(\v{x} \in \R^2\text{.}\)
    🔗
    Step 1: In this case, \(\v{e}_1 = \begin{bmatrix} 1 \\ 0 \end{bmatrix}\) and \(\v{e}_2 = \begin{bmatrix} 0 \\ 1 \end{bmatrix}\text{.}\) So
    \begin{equation*} T(\v{e}_1) = T\lp \begin{bmatrix} 1 \\ 0 \end{bmatrix} \rp = \begin{bmatrix} 2 \\ 5 \end{bmatrix} \qquad \text{ and } \qquad T(\v{e}_2) = T\lp \begin{bmatrix} 0 \\ 1 \end{bmatrix} \rp = \begin{bmatrix} 3 \\ 7 \end{bmatrix} \end{equation*}
    🔗
    Step 2: \(A = \begin{bmatrix} T(\v{e}_1)\amp T(\v{e}_2) \end{bmatrix} = \begin{bmatrix} 2 \amp 3 \\ 5 \amp 7 \end{bmatrix}\)
    🔗
    Let’s check to see if \(A\) really works!
    \begin{equation*} A\begin{bmatrix} x \\ y \end{bmatrix} = \begin{bmatrix} 2 \amp 3 \\ 5 \amp 7 \end{bmatrix} \begin{bmatrix} x \\ y \end{bmatrix} = \begin{bmatrix} 2x + 3y \\ 5x + 7y \end{bmatrix} \end{equation*}
    Woo!
    🔗
    🔗

    Checkpoint 1.8.3.

    Find the matrix that corresponds to the linear transformation \(T: \R^3 \to \R^2\) given by
    \begin{equation*} T\lp \begin{bmatrix} x \\ y \\ z \end{bmatrix} \rp = \begin{bmatrix} x + 3y + 7z \\ 3x + 5y - 8z \end{bmatrix} \end{equation*}
    🔗
    Solution.
    Step 1: In this case, since the domain is \(\R^3\text{,}\) our standard basis vectors are \(\v{e}_1 = \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix}\text{,}\) \(\v{e}_2 = \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix}\text{,}\) and \(\v{e}_3 = \begin{bmatrix} 0 \\ 0 \\ 1 \end{bmatrix}\text{.}\) Evaluating the transformation at each gives:
    \begin{equation*} T(\v{e}_1) = T\lp \begin{bmatrix} 1 \\ 0 \\ 0 \end{bmatrix} \rp = \begin{bmatrix} 1 \\ 3 \end{bmatrix}, \end{equation*}
    \begin{equation*} T(\v{e}_2) = T\lp \begin{bmatrix} 0 \\ 1 \\ 0 \end{bmatrix} \rp = \begin{bmatrix} 3 \\ 5 \end{bmatrix}, \end{equation*}
    \begin{equation*} T(\v{e}_3) = T\lp \begin{bmatrix} 0 \\ 0 \\ 1 \end{bmatrix} \rp = \begin{bmatrix} 7 \\ -8 \end{bmatrix} \end{equation*}
    🔗
    Step 2: We form the matrix by placing these resulting vectors into the columns.
    \begin{equation*} A = \begin{bmatrix} T(\v{e}_1)\amp T(\v{e}_2)\amp T(\v{e}_3) \end{bmatrix} = \begin{bmatrix} 1 \amp 3 \amp 7 \\ 3 \amp 5 \amp -8 \end{bmatrix} \end{equation*}
    🔗
    Let’s check to see if \(A\) really works!
    \begin{equation*} A\begin{bmatrix} x \\ y \\ z \end{bmatrix} = \begin{bmatrix} 1 \amp 3 \amp 7 \\ 3 \amp 5 \amp -8 \end{bmatrix} \begin{bmatrix} x \\ y \\ z \end{bmatrix} = \begin{bmatrix} x + 3y + 7z \\ 3x + 5y - 8z \end{bmatrix} \end{equation*}
    Yay!
    🔗
    🔗
    🔗
    🔗

    Subsection Transformation... What are we transforming?

    Have you wondered why we call \(T\) the linear transformation rather than just using the term "function" that we are more familiar with... Well for starter, "linear function" is not quite the same as linear transformation. Linear function is defined as \(f(x) = mx + b\) but this isn’t a linear transformation if \(b\) is not \(0\text{.}\) So using the term linear function in linear algebra may introduce ambiguity.
    🔗
    Moreover, each linear transformation can be interpreted as some sort of transformation in our usual geometry sense. So the term "transformation" here really means we are transforming some sort of objects. Let’s start by looking at an example!
    🔗

    Example 1.8.4.

    Suppose that \(A = \begin{bmatrix} 0 \amp 2 \\ 2 \amp 0 \end{bmatrix}\) and we define the linear transformation \(T: \R^2 \to \R^2\) by \(T(\v{x}) = A\v{x}\text{.}\) So \(T\) maps points on the plane to points on the plane.
    🔗
    Let’s figure out what this transformation does geometrically by doing the following:
    1. Draw a rectangle centered at \((0,0)\) and label the four vertices. This is your "before" picture.
      🔗
      🔗
    2. Plug in each of the four vertices using the rule \(T(\v{x}) = A\v{x}\) (write the points as column vectors and multiply!)
      🔗
      🔗
    3. Plot the four outputs and connect the dots.
      🔗
      🔗
    4. Describe what happened to your rectangle.
      🔗
      🔗
    🔗
    Solution. I will start with a rectangle with vertices \((1,-4)\text{,}\) \((1,4)\text{,}\) \((-1,-4)\text{,}\) and \((-1,4)\)
    First, let’s connect the dots to draw our rectangle. This is our "before" picture.
    🔗
    Plugging in each of the vertices to our transformation, we get
    \begin{align*} T\lp \begin{bmatrix} 1 \\-4 \end{bmatrix} \rp \amp= \begin{bmatrix} 0 \amp 2 \\ 2 \amp 0 \end{bmatrix} \begin{bmatrix} 1 \\ -4 \end{bmatrix} = \begin{bmatrix} -8 \\ 2 \end{bmatrix} \\ T\lp \begin{bmatrix} 1 \\4 \end{bmatrix} \rp \amp= \begin{bmatrix} 0 \amp 2 \\ 2 \amp 0 \end{bmatrix} \begin{bmatrix} 1 \\ 4 \end{bmatrix} = \begin{bmatrix} 8 \\ 2 \end{bmatrix} \\ T\lp \begin{bmatrix} -1 \\-4 \end{bmatrix} \rp \amp= \begin{bmatrix} 0 \amp 2 \\ 2 \amp 0 \end{bmatrix} \begin{bmatrix} -1 \\ -4 \end{bmatrix} = \begin{bmatrix} -8 \\ -2 \end{bmatrix} \\ T\lp \begin{bmatrix} -1 \\4 \end{bmatrix} \rp \amp= \begin{bmatrix} 0 \amp 2 \\ 2 \amp 0 \end{bmatrix} \begin{bmatrix} -1 \\ 4 \end{bmatrix} = \begin{bmatrix} 8 \\ -2 \end{bmatrix} \end{align*}
    Let’s graph the outputs and connect the dots. This is our "after" picture.
    🔗
    So what does the transformation do? It is a flip (across the line \(y = x\)) and a stretch (by a factor of \(2\))!
    🔗
    🔗
    🔗
    Your textbook includes the standard matrices for each type of transformation (page 102-104). For the completeness of this awesome notes, let’s include them below as well:
    🔗
    You may want to do a series of transformations (e.g., expand the object and flip it over). This can be achieved by composing multiple transformations. Matrix-wise, composing transformations means multiplying the corresponding standard matrix to the left. We will not be diving into composition in this class so feel free to explore more (and you may turn this into the class project if interest!).
    🔗
    There is one more type of the transformation that these tables did not include. That is the rotation. We can also do rotation using a linear transformation!
    🔗
    Let’s find the standard matrix for a rotation \(\theta\) in \(\R^2\text{!}\) We need to figure out what such a rotation does to the vectors \(\begin{bmatrix} 1 \\ 0 \end{bmatrix}\) and \(\begin{bmatrix} 0 \\ 1 \end{bmatrix}\text{.}\)
    🔗
    Let’s suppose that the vector \(\begin{bmatrix} x \\ y \end{bmatrix}\) is the result of rotating \(\begin{bmatrix} 1 \\ 0 \end{bmatrix}\) counterclockwise by the angle \(\theta\text{.}\) This vector can be represented as an arrow starting at the origin and ending at the point \((x, y)\text{.}\) We can draw a triangle to figure out the exact coordinates of the vector \(\begin{bmatrix} x \\ y \end{bmatrix}\text{.}\)
    🔗
    The \(x\)-coordinate will be \(\cos(\theta)\) and the \(y\)-coordinate will be \(\sin(\theta)\text{.}\) So the rotation takes \(\begin{bmatrix} 1 \\ 0 \end{bmatrix}\) to \(\begin{bmatrix} \cos(\theta) \\ \sin(\theta) \end{bmatrix}\text{.}\)
    🔗
    If we also draw the vector \(\begin{bmatrix} 0 \\ 1 \end{bmatrix}\) and rotate it (counterclockwise) by an angle of \(\theta\text{,}\) we get a picture like this:
    🔗
    This time, the \(x\)-coordinate will be \(-\sin(\theta)\) and the \(y\)-coordinate will be \(\cos(\theta)\text{.}\) So the rotation takes \(\begin{bmatrix} 0 \\ 1 \end{bmatrix}\) to \(\begin{bmatrix} -\sin(\theta) \\ \cos(\theta) \end{bmatrix}\text{.}\)
    🔗
    Note: These pictures make it easy to see this is true when \(\theta\) is between \(0^\circ\) and \(90^\circ\text{,}\) To see that this is true regardless of how big \(\theta\) is, you can either look at cases in all of the four quadrants or you can use trig identities.
    🔗
    This means that you can find any rotation matrix by plugging \(\theta\) into
    \begin{equation*} R_\theta = \begin{bmatrix} \cos(\theta) \amp -\sin(\theta) \\ \sin(\theta) \amp \cos(\theta) \end{bmatrix} \end{equation*}
    🔗
    Using this idea, we can figure out the standard matrices for more transformations! For example, you may be wondering how to reflect the object across an arbitrary line than the ones stated in the tables. Next, let’s find a matrix for the reflection across the line \(y = mx\text{!}\) That is, we need fo figure out what happens to \(\begin{bmatrix} 1 \\ 0 \end{bmatrix}\) and \(\begin{bmatrix} 0 \\ 1 \end{bmatrix}\) when we reflect them across the line \(y = mx\text{.}\)
    🔗
    From analytic geometry, we know that the slope of any line perpendicular to \(y = mx\) will have slope \(-\frac{1}{m}\text{.}\) We want to find the line through \((1,0)\) that has slope \(-\frac{1}{m}\text{.}\) That is because we need to find the intersection of that line with \(y = mx\) to get the midpoint of the segment between \((1,0)\) and the point we get after reflecting \((1,0)\text{.}\)
    🔗
    That perpendicular line is given by \(y - 0 = -\frac{1}{m}(x - 1)\text{,}\) which simplifies to \(y = -\frac{1}{m}x + \frac{1}{m}\text{.}\)
    🔗
    Now let’s find the point of intersection! Substituting \(y = mx\) into this equation, we get
    \begin{equation*} mx = -\frac{1}{m}x + \frac{1}{m} \qquad \implies \qquad m^2x = -x + 1 \end{equation*}
    This means that \(x = \frac{1}{1 + m^2}\text{.}\) Then substituting again, we get \(y = m\frac{1}{1 + m^2}\text{.}\) So the midpoint between \((1,0)\) and the result of reflecting \((1,0)\) across the line \(y = mx\) is \(\lp \frac{1}{1 + m^2}, m\frac{1}{1 + m^2} \rp\text{.}\)
    🔗
    The midpoint formula tells us that the midpoint of a segment between points \((x_1, y_1)\) and \((x_2, y_2)\) is \(\lp \frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2} \rp\text{.}\) So
    \begin{equation*} \lp \frac{1}{1 + m^2}, m\frac{1}{1 + m^2} \rp = \lp \frac{1 + x}{2}, \frac{0 + y}{2} \rp \end{equation*}
    where \((x,y)\) is the point we are looking for. This gives us the equations
    \begin{equation*} \begin{cases} \frac{1}{1 + m^2} = \frac{1 + x}{2} \\ m\frac{1}{1 + m^2} = \frac{y}{2} \end{cases} \end{equation*}
    Solving these equations, we get
    \begin{equation*} \begin{cases} x = \frac{1 - m^2}{1 + m^2} \\ y = \frac{2m}{1 + m^2} \end{cases} \end{equation*}
    So the reflection takes \((1,0)\) to \(\lp \frac{1 - m^2}{1 + m^2}, \frac{2m}{1 + m^2} \rp\text{.}\)
    🔗
    A similar method can be used to find out that the reflection takes the point \((0,1)\) to the point \(\lp \frac{2m}{1 + m^2}, \frac{m^2 - 1}{1 + m^2} \rp\text{.}\)
    🔗
    Converting these to column vectors, the matrix for reflecting across the line \(y = mx\) is given by
    \begin{equation*} R_m = \begin{bmatrix} \frac{1 - m^2}{1 + m^2} \amp \frac{2m}{1 + m^2} \\ \frac{2m}{1 + m^2} \amp \frac{m^2 - 1}{1 + m^2} \end{bmatrix} \end{equation*}
    Pulling out the common factor, we can rewrite this as
    \begin{equation*} R_m = \frac{1}{1 + m^2} \begin{bmatrix} 1 - m^2 \amp 2m \\ 2m \amp m^2 - 1 \end{bmatrix} \end{equation*}
    🔗
    🔗

    Subsection Special Types of Linear Transformations

    There are some special linear transformations, and we will focus on surjection and injection:
    🔗

    Definition 1.8.9.

    A mapping \(T: \R^n \to \R^m\) is a surjection (or onto \(\R^m\)) if each \(\v{b}\in \R^m\) is the image of at least one \(\v{x} \in \R^n\text{.}\)
    🔗
    That is, every element in the codomain of \(T\) is an image of the mapping.
    🔗
    🔗

    Definition 1.8.10.

    A mapping \(T: \R^n \to \R^m\) is an injection (or one-to-one) if each \(\v{b}\in \R^m\) is the image of at most one \(\v{x}\in \R^n\text{.}\)
    🔗
    That is, a mapping \(T\) is injective if \(T(\v{x}_1) = T(\v{x}_2)\) implies \(\v{x}_1 = \v{x}_2\text{.}\)
    🔗
    🔗
    Notice how parallel each of these definitions is. Moreover, these definitions have everything to do with the existence and uniqueness of pre-images:
    • Existence: Does each \(\v{b}\in \R^m\) have a pre-image? If \(T\) is surjective, then Yes.
      🔗
      🔗
    • Uniqueness: Is each solution to \(T(\v{x}) = \v{b}\) unique? If \(T\) is injective, then Yes!
      🔗
      🔗
    🔗
    An immediate (and famous) result about injection is the following theorem:
    🔗

    Theorem 1.8.11.

    Let \(T: \R^n \to \R^n\) be a linear transformation. Then \(T\) is injective if and only if \(T(\v{x}) = \v{0}\) has only the trivial solution.
    🔗
    🔗

    Proof.
    This is a biconditional statement. Proving this statement comes in to parts:
    🔗
    Forward direction (\(\Rightarrow\)): Assume that \(T\) is injective. We want to show that the equation \(T(\v{x}) = \v{0}\) has only the trivial solution.
    🔗
    Because \(T\) is a linear transformation, we know that it maps the zero vector to the zero vector, meaning \(T(\v{0}) = \v{0}\text{.}\) Suppose there is some vector \(\v{x}\) in \(\R^n\) such that \(T(\v{x}) = \v{0}\text{.}\) Then we have:
    \begin{equation*} T(\v{x}) = T(\v{0}) \end{equation*}
    Since we assumed \(T\) is injective, the outputs being equal guarantees that the inputs must also be equal. Therefore, \(\v{x} = \v{0}\text{.}\) This shows that the trivial solution is the only solution.
    🔗
    Backward direction (\(\Leftarrow\)): Assume that \(T(\v{x}) = \v{0}\) has only the trivial solution. We want to show that \(T\) is injective.
    🔗
    Suppose there are vectors \(\v{u}\) and \(\v{v}\) in \(\R^n\) such that \(T(\v{u}) = T(\v{v})\text{.}\) Subtracting \(T(\v{v})\) from both sides, we have
    \begin{equation*} T(\v{u}) - T(\v{v}) = \v{0} \end{equation*}
    By the linearity properties, this implies that
    \begin{equation*} T(\v{u} - \v{v}) = \v{0} \end{equation*}
    By our initial assumption, the only vector that maps to the zero vector is the zero vector itself. This forces the input to be zero. That is,
    \begin{equation*} \v{u} - \v{v} = \v{0} \end{equation*}
    This shows that \(\v{u} = \v{v}\text{,}\) and hence \(T\) is injective.
    🔗
    What we have established that these two claims are either both true or both false (we assume one of them is true, and show the other one is also true). Another way of calling biconditional statements are equivalence statement.
    🔗
    🔗
    Last but not least, let’s make a connection between these two types of the special transformations and what we have learned in the past through the following two statements!
    🔗

    Theorem 1.8.12.

    Let \(T: \R^n \to \R^m\) be a linear transformation with standard matrix \(A\text{.}\) Then \(T\) is surjective if and only if the columns of \(A\) span the codomain, \(\R^m\text{.}\)
    🔗
    🔗

    Proof.
    Observe that the theorem is a biconditional statement. But instead of proving both directions, let’s do a slick trick!
    🔗
    Notation clarification! The symbol \(\implies\) means implication!
    🔗
    Let’s consider the forward direction. The chain of implication looks like this:
    \begin{align*} \amp T \text{ is surjective} \\ \implies \qquad \amp \forall \v{b}\in \R^m, T(\v{x}) = \v{b} \text{ has at least one solution} \\ \implies \qquad \amp \forall \v{b}\in \R^m, A\v{x} = \v{b} \text{ is consistent} \\ \implies \qquad \amp \text{The columns of } A \text{ span } \R^m \end{align*}
    Okay. And this is supposed to be impressive how?
    🔗
    Observe that the statements here are all equivalent! That is, to prove the other direction, we can just reverse all the implication arrows and the argument stays true! So the proof of this theorem is to change all the implication arrow to a biconditional arrow!
    \begin{align*} \amp T \text{ is surjective} \\ \iff \qquad \amp \forall \v{b}\in \R^m, T(\v{x}) = \v{b} \text{ has at least one solution} \\ \iff \qquad \amp \forall \v{b}\in \R^m, A\v{x} = \v{b} \text{ is consistent} \\ \iff \qquad \amp \text{The columns of } A \text{ span } \R^m \end{align*}
    🔗
    🔗

    Theorem 1.8.13.

    Let \(T: \R^n \to \R^m\) be a linear transformation with standard matrix \(A\text{.}\) Then \(T\) is injective if and only if the columns of \(A\) are linearly independent.
    🔗
    🔗

    Proof.
    We can prove this theorem using the same slick trick!
    \begin{align*} \amp T \text{ is injective} \\ \iff \qquad \amp T(\v{x}) = \v{0} \text{ has has only the trivial solution} \\ \iff \qquad \amp A\v{x} = \v{0} \text{ has only the trivial solution} \\ \iff \qquad \amp \text{The columns of } A \text{ are linearly independent } \end{align*}
    🔗
    🔗

    Example 1.8.14.

    Let \(T: \R^2 \to \R^3\) defined by
    \begin{equation*} T\lp \begin{bmatrix} x_1 \\ x_2 \end{bmatrix} \rp = \begin{bmatrix} 3x_1 + x_2 \\ 5x_1 + 7x_2 \\ x_1 + 3x_2 \end{bmatrix} \end{equation*}
    Is \(T\) a linear transformation? Is it injective? Is it surjective?
    🔗
    First, we determine if \(T\) is a linear transformation. We can "factor out" the variables to rewrite the transformation as a matrix-vector product:
    \begin{equation*} T(\v{x}) = \begin{bmatrix} 3 \amp 1 \\ 5 \amp 7 \\ 1 \amp 3 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \end{bmatrix} \end{equation*}
    Let \(A\) be this \(3 \times 2\) matrix. Because \(T(\v{x}) = A\v{x}\text{,}\) it is a matrix transformation. Every matrix transformation is a linear transformation, so yes, \(T\) is a linear transformation.
    🔗
    Next, we check if \(T\) is injective. By our previous theorem, \(T\) is injective if and only if the homogeneous equation \(T(\v{x}) = \v{0}\) has only the trivial solution. To determine this, we find the reduced row echelon form of the standard matrix \(A\text{:}\)
    \begin{equation*} \begin{bmatrix} 3 \amp 1 \\ 5 \amp 7 \\ 1 \amp 3 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 3 \\ 5 \amp 7 \\ 3 \amp 1 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 3 \\ 0 \amp -8 \\ 0 \amp -8 \end{bmatrix} \sim \begin{bmatrix} 1 \amp 0 \\ 0 \amp 1 \\ 0 \amp 0 \end{bmatrix} \end{equation*}
    Because there is a pivot in every column, there are no free variables. The equation \(A\v{x} = \v{0}\) has only the trivial solution, \(\v{x} = \v{0}\text{.}\) Therefore, \(T\) is injective.
    🔗
    Finally, we determine if \(T\) is surjective. The transformation \(T\) is surjective if and only if its columns span the codomain, \(\R^3\text{.}\) This occurs if and only if there is a pivot position in every row of \(A\text{.}\) Looking at the reduced row echelon form of \(A\text{,}\) the third row does not contain a pivot. Consequently, the columns of \(A\) do not span \(\R^3\text{,}\) meaning there are vectors in the codomain that cannot be reached. Thus, \(T\) is not surjective.
