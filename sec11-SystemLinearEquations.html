    Section 1.1 Systems of Linear Equations

    Back in previous algebra classes, you probably learned how to solve system of linear equations algebraically using methods like substitution and elimination. While these methods are effective for small systems, they quickly become unwieldy as there are more variables and/or more equations.
    🔗
    As our introductory lesson to linear algebra, we will learn a more efficient method for solving systems of linear equations using elementary row operations on augmented matrices.
    🔗

    Objectives

    After this section, students will be able to:
    🔗
    • construct an augmented matrix from a system of linear equations (and vice versa);
      🔗
      🔗
    • use elementary row operations on an augmented matrix to solve a system of linear equations;
      🔗
      🔗
    • distinguish between systems with exactly one solution, infinitely many solutions, and no solution.
      🔗
      🔗
    • express the solution set of a system of linear equations with infinitely many solutions in parametric form.
      🔗
      🔗
    🔗

    Investigation 1.1.1.

    Recall we learned solving a (system of) linear equation(s) before. Let’s try solving some of them.
    🔗

    (a)

    The equation \(3x = 5\) is a linear equation. What is the solution(s) to this equation?
    🔗
    Solution.
    It is a system of linear equations. It’s just a boring one because there is only one variable and only one equation.
    🔗
    It has one solution, \(x = \dfrac{5}{3}\)
    🔗
    🔗
    🔗

    (b)

    The equation \(3x + 2y= 5\) is also a linear equation. What is the solution(s) to this equation?
    🔗
    Solution.
    This system has only one equation but it has two variables. It has infinitely many solutions!
    🔗
    Every point on the line \(y = \dfrac{-2}{2}x + \dfrac{5}{2}\) is a solution. Some specific solutions are \(\lp 0, \dfrac{5}{2} \rp\) and \((1,1)\text{.}\)
    🔗
    🔗
    🔗

    (c)

    The two linear equations \(3x = 5\) and \(2x = 7\) form a system of linear equations. What is the solution(s) to this system of equations?
    🔗
    Solution.
    This system has two equations but only one variable.
    🔗
    It doesn’t have any solutions! It isn’t possible for \(x\) to be both \(\dfrac{5}{3}\) and \(\dfrac{7}{2}\text{.}\)
    🔗
    🔗
    🔗
    🔗

    Subsection Solving Systems of Linear Equations

    The type of systems of linear equations you may encounter a lot before may look like this:
    \begin{equation*} \begin{cases} 3x + 2y = 5 \\ 5x - 8y = 14 \end{cases} \end{equation*}
    You probably learned the two most common ways to solve these equations.
    🔗
    The first method is called substitution. In this method, you will
    1. solve for one variable in terms of the other (using either equation);
      🔗
      🔗
    2. substitute the resulting expression in the other equation;
      🔗
      🔗
    3. substitute the resulting value into the expression from Step 1 (or into either equation) to get the second value.
      🔗
      🔗
    Let’s use this method to solve this system of linear equations!
    🔗
    If we solve the first equation for \(y\text{,}\) we get
    \begin{equation*} y = \frac{-3}{2} x + \frac{5}{2} \end{equation*}
    Then we can substitute this into the second equation to get
    \begin{equation*} 5x - 8 \lp \frac{-3}{2} x + \frac{5}{2} \rp = 14 \end{equation*}
    Then we can simplify and solve for \(x\) and get a specific number:
    \begin{align*} 5x - 8 \lp \frac{-3}{2} x + \frac{5}{2} \rp \amp= 14 \\ 8x + 12x - 20 \amp= 14 \\ 17x \amp= 34 \\ x \amp= 2 \end{align*}
    Then we can substitute this value of \(x\) into \(y = \frac{-3}{2}x + \frac{5}{2}\) to find the value of \(y\text{.}\)
    \begin{equation*} y = \frac{-3}{2} (2) + \frac{5}{2} = \frac{-1}{2} \end{equation*}
    So the solution to this system of linear equations is \(\lp 2, \dfrac{-1}{2} \rp\text{.}\)
    🔗
    NOTE: This method can be generalized and used for larger linear systems (more equations and/or more variables). But the next method is much better for larger systems.
    🔗
    You may have called the next method something like the "subtraction method" or the "elimination method". In this method you perform "elementary operations" on the equations until you get a system that is super easy to solve.
    🔗
    The elementary operations you can choose from are:
    1. swap two equations;
      🔗
      🔗
    2. multiply an equation by a constant (on both sides);
      🔗
      🔗
    3. add a multiple of one equation to the other equation.
      🔗
      🔗
    🔗
    Let’s use the elementary operations to solve the same system of linear equations as before:
    \begin{equation*} \begin{cases} 3x + 2y = 5 \\ 5x - 8y = 14 \end{cases} \end{equation*}
    🔗
    First, we will add \(4\) times the first equation to the second equation (replacing the second equation) to get
    \begin{equation*} \begin{cases} 3x + 2y = 5 \\ 17x = 34 \end{cases} \end{equation*}
    Next, we will multiply both sides of the second equation by \(\dfrac{1}{17}\) to get
    \begin{equation*} \begin{cases} 3x + 2y = 5 \\ x = 2 \end{cases} \end{equation*}
    Then we will add \(-3\) times the second equation to the first equation (replacing the first equation) to get
    \begin{equation*} \begin{cases} 2y = -1 \\ x = 2 \end{cases} \end{equation*}
    Finally, we will multiply the first equation by \(\dfrac{1}{2}\) to get
    \begin{equation*} \begin{cases} y = \dfrac{-1}{2} \\ x = 2 \end{cases} \end{equation*}
    Notice that we have now turned our system of equations into a new system of equations that is absurdly easy to solve. It literally just says the answer is \(\lp 2, \dfrac{-1}{2} \rp\text{.}\)
    🔗
    But why does the method of elementary operations work? This method works because none of the three elementary operations changes the solution set of the system of equations. At each step, the set of solutions of the system before you perform the elementary operation is exactly the same as the solution set after you perform the elementary operation.
    🔗

    Note 1.1.1. But Richard... Why don’t the elementary operations change the solution set of a system of equations?

    Instead of giving a formal proof for all cases (which you can find in the book if you want), Richard will explain why it works in the case of a system of two equations with two variables:
    1. It is obvious that swapping equations clearly can’t change the solution set. It makes no difference what order the equations are listed in.
      🔗
      🔗
    2. It is also pretty clear that multiplying both sides of an equation by a non-zero number will not change the solution set.
      🔗
      First, notice that if \((x_0, y_0)\) is a solution before, then it is a solution after. That is, if \(ax_0 + by_0 = c\text{,}\) then it is also true that \(k(ax_0 + by_0) = k(c)\) for any non-zero constant \(k\text{.}\) This is because when two numbers are equal, you get the same answer when you multiply each of them by the same number.
      🔗
      Second, notice that if \((x_0,y_0)\) is a solution after, then it is a solution before. Since the number \(k\) is not zero, \(\frac{1}{k}\) exists and is not zero. So if \(k(ax_0 + by_0) = k(c)\text{,}\) then it is also true that \(ax_0 + by_0 = c\) (you can just multiply both sides by \(\frac{1}{k}\text{!}\)).
      🔗
      🔗
    3. The least obvious one is the elementary operation of adding a multiple of one equation to another equation (and replacing the equation). We can think of this elementary operation as doing two steps:
      1. multiply one equation by a non-zero constant, and
        🔗
        🔗
      2. add one equation to another equation (and replace that equation with the sum).
        🔗
        🔗
      The first step is just the operation of multiplying an equation by a non-zero constant, which we already know does not change the solution set. So we just need to demonstrate that adding one equation to another (and replacing the second equation with the sum) does not change the solution set.
      🔗
      First, we need to show that anything that was a solution before is still a solution after. Consider the system of linear equations
      \begin{equation*} \begin{cases} a_1x + b_1y = c_1 \\ a_2x + b_2y = c_2 \end{cases} \end{equation*}
      Suppose that \(a_1x_0 + b_1y_0 = c_1\) and \(a_2x_0 + b_2y_0 = c_2\) (in other words suppose \((x_0,y_0)\) is a solution). We want to show that \((x_0,y_0)\) is still a solution to the system
      \begin{equation*} \begin{cases} a_1x + b_1y = c_1 \\ (a_2 + a_1)x + (b_2 + b_1)y = c_2 + c_1 \end{cases} \end{equation*}
      Since we already know that \((x_0,y_0)\) is a solution to the first equation, we just need to show it is a solution to the second equation.
      🔗
      Since the number \(a_2x_0 + b_2y_0\) is equal to the number \(c_2\text{.}\) If we add \(a_1x_0 + b_1y_0\) to both of them, we get the same thing. So we get
      \begin{equation*} a_1x_0 + b_1y_0 + a_2x_0 + b_2y_0 = a_1x_0 + b_1y_0 + c_2 \end{equation*}
      And since we know that \(a_1x_0 + b_1y_0 = c_1\text{,}\) we can substitute to get
      \begin{equation*} a_1x_0 + b_1y_0 + a_2x_0 + b_2y_0 = c_1 + c_2 \end{equation*}
      Do some simplifying and factoring on the left side and we get
      \begin{equation*} (a_2 + a_1)x_0 + (b_2 + b_1)y_0 = c_1 + c_2 \end{equation*}
      This proves \((x_0,y_0)\) is indeed a solution to the second equation!
      🔗
      Second, we want to show that if \((x_0,y_0)\) is a solution after then it is a solution before. But really we already have! That is because we can get from the after system to the before system by multiplying the first equation by \(-1\) and adding it to the second equation. So going backwards just involves doing the exact same thing as going forward (just after multiplying by a non-zero constant which we know always works!).
      🔗
      🔗
    🔗
    Note 1.1.2. But Richard... Why bother showing the solution sets are the same before and after, and vice versa?
    That is because this establishes that the elementary operations are reversible! Then we can call the two systems "equivalent" in the sense that they have the same solution set.
    🔗
    To give you an example of non-reversible operations, consider the operation of "squaring both sides of an equation".
    🔗
    Let’s say we have an equation \(x = 2\text{.}\) Obviously, there is only one solution to this equation, which is \(x = 2\text{.}\) If we apply this operation, then we get the equation \(x^2 = 4\text{.}\) Clearly, \(x = 2\) is still a solution to this new equation. Hence, in this operation, the solution before is still a solution after.
    🔗
    Yet, the new equation has another solution, which is \(x = -2\text{.}\) This isn’t a solution to the original equation, and hence, the solution after may not be a solution before.
    🔗
    This shows that this operation is not reversible and hence the two equations are not equivalent (in the sense that they don’t have the same solution set).
    🔗
    🔗
    🔗

    Checkpoint 1.1.3.

    Use the elementary operations to solve the following system of linear equations:
    \begin{equation*} \begin{cases} 6x - 3y = 3 \\ 5x + 2y = -11 \end{cases} \end{equation*}
    🔗
    Solution.
    There are a lot of ways to do this. This is one:
    🔗
    Step 1: Multiply the second equation by \(3\) to get
    \begin{equation*} \begin{cases} 6x - 3y = 3 \\ 15x + 6y = -33 \end{cases} \end{equation*}
    🔗
    Step 2: Add \(2\) times the first equation to the second equation (replacing the second equation) to get
    \begin{equation*} \begin{cases} 6x - 3y = 3 \\ 27x = -27 \end{cases} \end{equation*}
    🔗
    Step 3: Multiply the second equation by \(\dfrac{1}{27}\) to get
    \begin{equation*} \begin{cases} 6x - 3y = 3 \\ x = -1 \end{cases} \end{equation*}
    🔗
    Step 4: Add \(-6\) times the second equation to the first equation (replacing the first equation) to get
    \begin{equation*} \begin{cases} -3y = 9 \\ x = -1 \end{cases} \end{equation*}
    🔗
    Step 5: Multiply the first equation by \(\dfrac{-1}{3}\) to get
    \begin{equation*} \begin{cases} y = -3 \\ x = -1 \end{cases} \end{equation*}
    🔗
    And that’s our solution, \((-1,-3)\text{!}\)
    🔗
    🔗
    🔗
    🔗

    Subsection Augmented Matrices and Elementary Operations

    Basically, a matrix is just an array of numbers inside of a set of brackets. For example, \(\begin{bmatrix}3 \amp 2 \\ 5 \amp -8 \end{bmatrix}\) is a \(2 \times 2\) matrix since there are \(2\) rows and \(2\) columns.
    🔗
    We will learn a lot more about matrices this term! But first we will learn about a kind of "fake" matrix called an augmented matrix. An augmented matrix is compressed version of a matrix equation which we will learn about soon.
    🔗
    For any system of equations we can make the associated augmented matrix by basically writing down the equation inside of a big set of brackets and erasing all of the variables and addition/subtraction signs and replacing all of the equal signs with a big vertical line (some people used a dashed line).
    🔗
    For example, we can convert the following system of linear equations
    \begin{equation*} \begin{cases} 3x + 2y = 5 \\ 5x - 8y = 14 \end{cases} \end{equation*}
    into its associated augmented matrix as
    \begin{equation*} \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 5 \amp -8 \amp \mid \amp 14 \end{bmatrix} \end{equation*}
    The part of the augmented matrix to the left of the vertical line(s) is called the coefficient matrix (the coefficient part of the system) and the part to the right of the vertical line(s) is called the constant matrix (the constant part of the system).
    🔗
    When you write a system of linear equations as an augmented matrix, you can do the exact same operations to your equations except now you do them to the rows of the matrix (each row is just a compressed version of an equation!). So the three elementary row operations are:
    1. swap two rows;
      🔗
      🔗
    2. multiply a row by a non-zero constant;
      🔗
      🔗
    3. add a multiple of a row to another row (replacing that row).
      🔗
      🔗
    You can solve a system of equations by (1) writing it as an augmented matrix, (2) performing elementary row operations until it is in a "nice" form, and then (3) writing this nice version as a system of linear equations (and reading off the answer).
    🔗
    Let’s solve the previous system of linear equations using the augmented matrix. Our augmented matrix is
    \begin{equation*} \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 5 \amp -8 \amp \mid \amp 14 \end{bmatrix} \end{equation*}
    We can add \(4\) times Row 1 to Row 2 (replacing Row 2) to get
    \begin{equation*} \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 17 \amp 0 \amp \mid \amp 34 \end{bmatrix} \end{equation*}
    Then we can multiply Row 2 by \(\dfrac{1}{17}\) to get
    \begin{equation*} \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 1 \amp 0 \amp \mid \amp 2 \end{bmatrix} \end{equation*}
    And then add \(-3\) times Row 2 to Row 1 (replacing Row 1) to get
    \begin{equation*} \begin{bmatrix} 0 \amp 2 \amp \mid \amp -1 \\ 1 \amp 0 \amp \mid \amp 2 \end{bmatrix} \end{equation*}
    Finally, we can multiply Row 1 by \(\dfrac{1}{2}\) to get
    \begin{equation*} \begin{bmatrix} 0 \amp 1 \amp \mid \amp \dfrac{-1}{2} \\ 1 \amp 0 \amp \mid \amp 2 \end{bmatrix} \end{equation*}
    Now if we convert this back to a system of linear equations, we get
    \begin{equation*} \begin{cases} y = \dfrac{-1}{2} \\ x = 2 \end{cases} \end{equation*}
    And it’s already solved!
    🔗

    Checkpoint 1.1.4.

    Solve the following system of linear equations (again).
    \begin{equation*} \begin{cases} 6x - 3y = 3 \\ 5x + 2y = -11 \end{cases} \end{equation*}
    This time make the augmented matrix and use elementary row operations to solve it.
    🔗
    Solution.
    The augmented matrix for this system is
    \begin{equation*} \begin{bmatrix} 6 \amp -3 \amp \mid \amp 3 \\ 5 \amp 2 \amp \mid \amp -11 \end{bmatrix} \end{equation*}
    There are a lot of ways to solve the system. Here is one:
    🔗
    Step 1: Multiply Row 2 by \(3\) to get
    \begin{equation*} \begin{bmatrix} 6 \amp -3 \amp \mid \amp 3 \\ 15 \amp 6 \amp \mid \amp -33 \end{bmatrix} \end{equation*}
    🔗
    Step 2: Add \(2\) times Row 1 to Row 2 (replacing Row 2) to get
    \begin{equation*} \begin{bmatrix} 6 \amp -3 \amp \mid \amp 3 \\ 27 \amp 0 \amp \mid \amp -27 \end{bmatrix} \end{equation*}
    🔗
    Step 3: Multiply Row 2 by \(\dfrac{1}{27}\) to get
    \begin{equation*} \begin{bmatrix} 6 \amp -3 \amp \mid \amp 3 \\ 1 \amp 0 \amp \mid \amp -1 \end{bmatrix} \end{equation*}
    🔗
    Step 4: Add \(-6\) times Row 2 to Row 1 (replacing Row 1) to get
    \begin{equation*} \begin{bmatrix} 0 \amp -3 \amp \mid \amp 9 \\ 1 \amp 0 \amp \mid \amp -1 \end{bmatrix} \end{equation*}
    🔗
    Step 5: Multiply Row 1 by \(\dfrac{-1}{3}\) to get
    \begin{equation*} \begin{bmatrix} 0 \amp 1 \amp \mid \amp -3 \\ 1 \amp 0 \amp \mid \amp -1 \end{bmatrix} \end{equation*}
    🔗
    Then we can write this as a system of equations again and get
    \begin{equation*} \begin{cases} y = -3 \\ x = -1 \end{cases} \end{equation*}
    And that’s our solution, \((-1,-3)\text{!}\)
    🔗

    Subsection How many solutions can a linear system have?

    The system of linear equations in the previous example has exactly one solution (which we found in three different ways!) but not all linear systems have exactly one solution. Some do not have any solutions and some have infinitely many solutions.
    🔗
    Let’s solve the following system of linear equations using the augmented matrix and elementary row operations:
    \begin{equation*} \begin{cases} 3x + 2y = 5 \\ -6x - 4y = 7 \end{cases} \end{equation*}
    The augmented matrix is
    \begin{equation*} \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ -6 \amp -4 \amp \mid \amp 7 \end{bmatrix} \end{equation*}
    Let’s first add \(2\) times Row 1 to Row 2 (replacing Row 2). We get
    \begin{equation*} \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 0 \amp 0 \amp \mid \amp 17 \end{bmatrix} \end{equation*}
    Now we can write this as a system of linear equations, and we get
    \begin{equation*} \begin{cases} 3x + 2y = 5 \\ 0 = 17 \end{cases} \end{equation*}
    But... \(0\) is definitely NOT equal to \(17\text{.}\) So this system has no solution!
    🔗
    Notice that both of the equations in our original system are lines (that’s where the "linear" comes from!). Richard graphed these two lines and they look like this:
    🔗
    These lines are parallel and they don’t intersect!
    🔗
    When you have three variables then each equation is a plane, so you get no solutions when one of the planes is parallel to (but not equal to) the other. And the same kind of thing happens in higher dimensions except that it would blow your mind if you tried to see it!
    🔗
    What happens in your augmented matrix when you have no solution? You zero out a row of coefficients using elementary operations BUT the constant in that row is NOT zero.
    🔗
    Next, let’s solve this system of linear equations using the augmented matrix and elementary row operations:
    \begin{equation*} \begin{cases} 3x + 2y = 5 \\ -6x - 4y = -10 \end{cases} \end{equation*}
    The augmented matrix is
    \begin{equation*} \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ -6 \amp -4 \amp \mid \amp -10 \end{bmatrix} \end{equation*}
    This time, we will add \(2\) times Row 1 to Row 2 (replacing Row 2), and we get
    \begin{equation*} \begin{bmatrix} 3 \amp 2 \amp \mid \amp 5 \\ 0 \amp 0 \amp \mid \amp 0 \end{bmatrix} \end{equation*}
    Now we can write this as a system of linear equations. We get
    \begin{equation*} \begin{cases} 3x + 2y = 5 \\ 0 = 0 \end{cases} \end{equation*}
    This time, the second equation is true. \(0\) IS definitely equal to \(0\text{.}\) Notice that our second equation is true no matter what \(x\) and \(y\)are! That means that any \((x,y)\) that satisfies the first equation will satisfy both equations. So the solution is every point on the line \(3x + 2y = 5\text{.}\)
    🔗
    If you were to graph the two original equations, this time you would find that they were actually the exact same line!
    🔗
    When you have infinitely many solutions, you will want to write the solution set in parametric form. The way to do this (for a two-variable system) is to solve for one variable in terms of the other and then write a general form for any solution. For this example, we can solve for \(y\) in terms of \(x\) and get
    \begin{equation*} y = \frac{-3}{2} x + \frac{5}{2} \end{equation*}
    So for any value of \(x\text{,}\) we get the solution \(\lp x, \dfrac{-3}{2} x + \dfrac{5}{2} \rp\text{.}\) In this case, the \(x\) is called the parameter.
    🔗
    We can pick any value of \(x\) we want and plug it into that formula to get a solution to the system. For example, if Richard picks \(x = 10\text{,}\) then he will get the solution \(\lp 10, \frac{-25}{2} \rp\text{.}\)
    🔗

    Note 1.1.6. Notation Warning!

    Some textbooks like to introduce a new letter to use for the parameter. For example, in this case they would probably use \(t\) and write the solutions as
    \begin{equation*} \lp t, \frac{-3}{2}t + \frac{5}{2} \rp \end{equation*}
    This is unnecessary but Richard will do it from time to time just to make sure you understand where these mysterious parameters come from.
    🔗
    🔗
    Sometimes we may need more than one parameter to describe all of the solutions to a system of equations. But that is a story for another day (that day will be Thursday).
    🔗
    What happens in your augmented matrix when you have infinitely many solutions? You zero out a row of coefficients using elementary operations AND the constant in that row is also zero.
    🔗
    🔗

    Subsection Existence and Uniqueness Questions

    Recall that a system of linear equations can have either no solution, exactly one solution, or infinitely many solutions. We ask two questions to determine which of these three cases we are in:
    1. Is the system consistent? That is, does a solution exist?
      🔗
      🔗
    2. If a solution exists, is it the only one? That is, is the solution unique?
      🔗
      🔗
    🔗
    Right now, we can answer these questions by actually solving the system of equations and seeing what happens. In Section 1.2, we will learn how to answer these questions by using only the augmented matrix and elementary row operations without having to write the system of equations out.
